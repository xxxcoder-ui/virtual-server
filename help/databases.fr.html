Modifier base de données
Cette page liste tous les bases de données qui sont actuellement détenues par le serveur virtuel, et permet la création de plus bases de données jusqu'à une limite en vigueur pour l'utilisateur connecté à Virtualmin. Vous pouvez également supprimer un plusieurs bases de données en les sélectionnant dans la liste et en appuyant sur le bouton Supprimer la sélection.

Pour voir les tableaux et les données réelles dans une base de données, cliquez sur le lien Gérer à côté du nom de base de données dans la table. Pour voir la taille de la base de données, cliquez sur son nom.
