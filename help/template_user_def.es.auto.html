¿Agregar usuario de Apache al grupo Unix para nuevos servidores?
Esta opción determina si el usuario con el que se ejecuta Apache se agregará al nuevo grupo de host virtual. Cuando está habilitado, los permisos se pueden ajustar significativamente en los directorios de inicio de los usuarios. Cuando los permisos más estrictos (es decir, 750) se combinan con SuExec, hace que sea imposible para los propietarios de dominios ver otros directorios y archivos de propietarios de dominios. Esta opción se recomienda en cualquier plataforma que admita grupos secundarios.
