Pode editar servidor virtual
Permite editar recursos do servidor virtual, sua senha e descrição.
Pode gerenciar usuários
Permite que usuários de caixas de correio, FTP e banco de dados sejam visualizados, editados, criados e excluídos.
Pode gerenciar aliases
Permite que os aliases de email sejam visualizados, editados, criados e excluídos.
Pode gerenciar bancos de dados
Permite que bancos de dados MySQL e PostgreSQL adicionais sejam adicionados a este servidor virtual ou excluídos dele.
Pode instalar scripts
Permite que scripts de terceiros sejam instalados neste servidor virtual.
Pode mudar o endereço IP
Se este servidor tiver um endereço IP privado, essa opção permitirá que ele seja alterado.
Certificados SSL gerenciados
Permite que o certificado SSL e a chave desse servidor sejam carregados ou gerados novamente.
Pode editar o quadro de encaminhamento
Para sites somente para encaminhamento, essa opção permite que o destino e o HTML do quadro de encaminhamento sejam editados.
Pode gerenciar administradores extras
Permite que contas extras de administrador do proprietário do servidor sejam visualizadas, criadas, editadas e excluídas.
Pode configurar a entrega de spam e vírus
Permite editar destinos e pontuações de entrega de spam e vírus.
Pode selecionar versões do PHP
Para domínios com PHP executados via CGI ou fastCGI, permite definir versões por diretório do PHP.
Pode editar configurações de email
Permite a configuração de BCCing automático, modo de cópia alternativo e outras opções relacionadas a email.
Pode fazer backups
Permite fazer backup de servidores virtuais proprietários em um servidor FTP ou SSH remoto.
Pode agendar backups
Permite fazer backup de servidores virtuais regularmente.
Pode restaurar backups
Permite restaurar o diretório inicial e o conteúdo do banco de dados dos servidores virtuais próprios.
Pode selecionar IPs compartilhados
Permite que endereços IP compartilhados definidos globalmente sejam selecionados ao criar servidores virtuais.
Pode criar aliases catchall
Permite a criação de aliases de email que encaminham todos os emails para um domínio para um domínio ou endereço diferente.
Pode usar o editor de páginas da web
Permite que as páginas da web sejam editadas via interface do usuário do Virtualmin.
Pode editar hosts remotos do banco de dados
Permite a entrada de sistemas clientes remotos com permissão para se conectar aos bancos de dados MySQL do domínio.
Pode desativar servidores virtuais
Permite que os servidores sejam desativados temporariamente.
Pode excluir servidores virtuais
Permite que os servidores proprietários sejam completamente excluídos, incluindo todos os sub-servidores e aliases.
