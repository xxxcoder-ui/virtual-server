Autorizzazioni per la directory del sito Web
Questa opzione specifica le autorizzazioni iniziali per i file e le directory all'interno della directory del sito Web specificata nell'opzione precedente. Se SuExec è stato abilitato e l'utente Apache viene aggiunto al gruppo utenti del server virtuale, queste autorizzazioni possono essere rafforzate per impedire ad altri utenti di vedere il contenuto della directory del sito Web. Questo insieme ristretto di autorizzazioni è consigliato per qualsiasi distribuzione a meno che tutti gli utenti con accesso alla shell siano utenti fidati. Le autorizzazioni consigliate sono 750, che equivale a rwxr-x ---. In altre parole, il proprietario può leggere, scrivere ed eseguire, il gruppo può leggere ed eseguire e altri non dispongono delle autorizzazioni per leggere, scrivere o eseguire i file all'interno della directory.
