Formato de dirección para hosts virtuales Apache
Esta opción determina el formato de dirección que utilizará Virtualmin en los nuevos bloques Apache <Virtualhost>. Una estrella hará coincidir las solicitudes que llegan a cualquier dirección, mientras que una IP solo hará coincidir las solicitudes con esa dirección específica. El comportamiento predeterminado se determina mirando la versión de Apache y qué directivas NameVirtualHost existen, si las hay.
