Capacités d'édition par défaut
Ces cases à cocher déterminent quelles capacités de leurs propriétaires de domaine de serveurs virtuels peuvent modifier par défaut. Une fois un serveur créé, ceux-ci peuvent être modifiés sur la page Modifier les limites du propriétaire.

Si l'option Automatique est sélectionnée, les limites sont déterminées selon que le propriétaire du serveur virtuel est autorisé ou non à créer des sous-serveurs (contrôlé par le champ Limiter le nombre de serveurs virtuels ). Si c'est le cas, il aura accès à toutes les capacités. Sinon, il ne peut gérer que les utilisateurs, les alias et modifier les pages Web.
