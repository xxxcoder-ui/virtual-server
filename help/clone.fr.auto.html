Cloner un serveur virtuel
Cette page vous permet de dupliquer un serveur virtuel existant avec un nouveau nom de domaine. Tout contenu Web, enregistrements DNS, boîtes aux lettres, alias de messagerie, bases de données et autres paramètres associés au domaine d'origine seront dupliqués, si possible. Cependant, le contenu Web ne sera en aucun cas modifié, donc si le serveur virtuel contient une application Web qui a été configurée pour utiliser les bases de données ou le répertoire de base de l'ancien serveur, il devra être reconfiguré manuellement après le clonage.

Lors de la duplication d'un serveur virtuel de niveau supérieur, vous devez également entrer un nouveau nom d'utilisateur et mot de passe pour posséder le nouveau domaine. Si le serveur a une adresse IP privée, une nouvelle IP sera automatiquement allouée au clone.

Le processus de clonage copiera toutes les bases de données associées au domaine d'origine sous de nouveaux noms et dupliquera leur contenu. Cependant, il ne mettra PAS à jour les scripts installés dans le domaine d'origine qui sont configurés pour utiliser les bases de données d'origine - cela doit être fait manuellement.
