Ruby script uitvoer mode
Deze optie bepaald of de Ruby scripts worden uitgevoerd als de Apache web server gebruiker (typisch httpd of apache), of de Unix gebruiker waar de virtuele server van is. De vorige optie is sneller omdat er geen extra processen hoeven worden opgestart om de Ruby code uit te voeren aar het is minder veilig omdat scripts files lezen en beschrijven als een gebruiker en niet als een domein eigenaar.
