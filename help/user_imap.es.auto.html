Inicio de sesión IMAP / FTP
Este campo muestra el nombre de usuario que este buzón deberá usar cuando inicie sesión a través de IMAP para recoger correo electrónico, SMTP para enviar correo o FTP para administrar archivos. Difiere de la primera parte de su dirección de correo electrónico, ya que tiene algunos o todos los nombres de dominio agregados o anexados.
