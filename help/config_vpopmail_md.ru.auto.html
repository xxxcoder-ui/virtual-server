Подкаталог VPOPMail для электронной почты
Это поле позволяет вам настроить подкаталог в каждом доме пользователя VPOPMail, который будет содержать электронную почту. В системах, которые не используют Maildir по умолчанию, его необходимо изменить, чтобы он соответствовал фактическому пути, используемому VPOPMail - например, в Gentoo Linux используется .maildir .
