Pode selecionar entre endereços IP compartilhados?
Quando essa opção é definida como Sim e existem vários endereços IP compartilhados no sistema, o revendedor poderá selecionar um ao criar um novo servidor virtual. Caso contrário, ele ficará limitado aos IPs compartilhados selecionados nos campos acima ou se nenhum for escolhido para o endereço padrão do sistema global.
