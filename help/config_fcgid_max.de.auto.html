Maximale Ausführungszeit von fcgid
Standardmäßig legt Virtualmin die maximale Ausführungszeit für fastCGI-Prozesse so fest, dass sie mit der Ausführungszeit von PHP-Skripten übereinstimmt, wie in der Datei php.ini der Domäne php.ini . Dies stellt sicher, dass PHP-Anwendungen nicht länger als vorgesehen ausgeführt werden.

Sie können dieses Verhalten jedoch deaktivieren, indem Sie in diesem Konfigurationsfeld Nicht festlegen auswählen. Oder Sie können Ihr eigenes Zeitlimit für fastCGI-Skripte angeben, indem Sie Zeit in Sekunden auswählen und eine Zahl eingeben.

Dies wird implementiert, indem die Apache IPCCommTimeout Direktive oder das Äquivalent in Nginx festgelegt wird.
