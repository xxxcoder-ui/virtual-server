Limiti del proprietario del server
Questa pagina consente all'amministratore principale o ad un rivenditore di limitare ciò che il proprietario di un server virtuale può fare usando Virtualmin. È possibile limitare il numero di domini, cassette postali, alias e altri oggetti che può creare e controllare quali funzionalità di un server virtuale è in grado di gestire.

Queste restrizioni e limiti si applicano a tutti i server secondari nel server di appartenenza selezionato. I limiti al numero di cassette postali, alias e database includono tutti i server, anziché essere applicati su base per server. Questi limiti non si applicano all'amministratore principale o ai rivenditori, ma solo ai proprietari dei server e ai loro account amministratore aggiuntivi.

Se un server è di proprietà di un rivenditore che ha limiti su domini, cassette postali o alias, verranno applicati anche i limiti impostati in questa pagina. Il più basso di ciascun limite è quello che avrà effetto.
