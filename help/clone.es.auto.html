Clonar servidor virtual
Esta página le permite duplicar un servidor virtual existente con un nuevo nombre de dominio. Cualquier contenido web, registros DNS, buzones, alias de correo, bases de datos y otras configuraciones asociadas con el dominio original se duplicarán, siempre que sea posible. Sin embargo, el contenido web no se cambiará de ninguna manera, por lo que si el servidor virtual contiene una aplicación web que se ha configurado para usar las bases de datos o el directorio de inicio del servidor anterior, deberá volver a configurarse manualmente después de la clonación.

Al duplicar un servidor virtual de nivel superior, también debe ingresar un nuevo nombre de usuario y contraseña para ser propietario del nuevo dominio. Si el servidor tiene una dirección IP privada, se asignará automáticamente una nueva IP para el clon.

El proceso de clonación copiará las bases de datos asociadas con el dominio original con nuevos nombres y duplicará sus contenidos. Sin embargo, NO actualizará ningún script instalado en el dominio original que esté configurado para usar las bases de datos originales, esto debe hacerse manualmente.
