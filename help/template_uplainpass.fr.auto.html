Stocker des mots de passe en texte clair?
Lorsque cette option est définie sur Oui , Virtualmin stockera les mots de passe en texte clair pour les propriétaires de domaine et les boîtes aux lettres. Cela leur permet d'être affichés dans l'interface utilisateur, mais ajoute un risque de sécurité potentiel si votre système est compromis. Si vous sélectionnez Non , tous les nouveaux serveurs virtuels créés à partir de ce modèle ne stockeront à la place que des mots de passe hachés unidirectionnels.

Étant donné que les applications Web doivent stocker les mots de passe de base de données en texte brut, le mot de passe de base de données pour un utilisateur sera défini sur un nouveau mot de passe aléatoire si Virtualmin est défini pour ne pas stocker les mots de passe en texte clair. Vous pouvez afficher ce mot de passe en accédant à Modifier les bases de données - <Mots de passe.
