Ajouter un utilisateur Apache au groupe Unix pour de nouveaux serveurs?
Cette option détermine si l'utilisateur sous lequel Apache s'exécute sera ajouté au nouveau groupe d'hôtes virtuel. Lorsqu'elle est activée, les autorisations peuvent être considérablement resserrées sur les répertoires personnels des utilisateurs. Lorsque des autorisations plus strictes (par exemple 750) sont combinées avec SuExec, il est impossible pour les propriétaires de domaine de voir les autres répertoires et fichiers des propriétaires de domaine. Cette option est recommandée sur toute plate-forme prenant en charge les groupes secondaires.
