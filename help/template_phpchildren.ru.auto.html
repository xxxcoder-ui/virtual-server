Подпроцессы FCGId PHP по умолчанию
Когда PHP-скрипты запускаются через FCGId, в каждом домене будет постоянно установлено количество PHP-процессов, установленных в этом поле для обслуживания запросов. Вы можете увеличить это значение по умолчанию, равное 4, чтобы увеличить задержку PHP-скрипта, или уменьшить его, чтобы уменьшить использование памяти.

Если установить значение None , процессы PHP будут запускаться только по мере необходимости и очищаться после некоторого периода бездействия.
