Permissões de login
Este menu determina que tipo de acesso de login é concedido ao usuário. Normalmente, você poderá escolher apenas email ou email e FTP, mas as opções podem ser personalizadas pelo administrador principal.

O nível de acesso selecionado realmente define o shell para o usuário Unix subjacente, que é usado pelos servidores FTP e SSH para determinar se ele pode fazer login ou não.
