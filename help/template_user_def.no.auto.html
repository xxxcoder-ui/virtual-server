Vil du legge til Apache-bruker i Unix-gruppen for nye servere?
Dette alternativet avgjør om brukeren som Apache kjører som skal legges til den nye virtuelle vertsgruppen. Når dette er aktivert, kan tillatelsene strammes betydelig i hjemmekatalogene. Når strammere tillatelser (dvs. 750) kombineres med SuExec, gjør det det umulig for domeneeiere å se andre kataloger og filer for domeneeiere. Dette alternativet anbefales på alle plattformer som støtter sekundære grupper.
