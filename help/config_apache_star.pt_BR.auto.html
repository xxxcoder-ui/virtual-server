Formato de endereço para hosts virtuais Apache
Esta opção determina o formato de endereço que o Virtualmin usará nos novos blocos Apache <Virtualhost>. Uma estrela corresponderá a solicitações que chegam a qualquer endereço, enquanto um IP corresponderá apenas a solicitações desse endereço específico. O comportamento padrão é determinado observando a versão do Apache e quais diretivas NameVirtualHost existem, se houver.
