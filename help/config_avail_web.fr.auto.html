Serveur Web Apache (pour l'hôte virtuel)
Activez cette option si le compte propriétaire du domaine doit avoir la possibilité de modifier la configuration Apache pour son domaine. Les privilèges accordés par cette option peuvent être intimidants pour les utilisateurs inexpérimentés, mais les utilisateurs expérimentés trouveront la flexibilité très puissante. Il est également possible, grâce à des modifications de la configuration de contrôle d'accès par défaut, d'autoriser l'accès direct à la section des domaines virtuels du fichier de configuration pour Apache. Des protections sont en place pour empêcher les comportements malveillants, mais il est peu probable qu'elles soient parfaites. Par défaut, les privilèges accordés sont généralement sûrs pour une utilisation dans un environnement d'hébergement partagé.
