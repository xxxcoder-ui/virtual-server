Subprocesos PHP FCGId predeterminados
Cuando los scripts PHP se ejecutan a través de FCGId, cada dominio tendrá la cantidad de procesos PHP configurados en este campo ejecutándose en todo momento para atender solicitudes. Puede aumentar esto desde el valor predeterminado de 4 para mejorar la latencia del script PHP, o disminuirlo para reducir el uso de memoria.

Si lo configura en Ninguno , los procesos PHP se iniciarán solo cuando sea necesario a pedido y se limpiarán después de un período de inactividad.
