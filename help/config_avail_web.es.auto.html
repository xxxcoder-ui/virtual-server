Servidor web Apache (para host virtual)
Habilítelo si la cuenta del propietario del dominio debe poder editar la configuración de Apache para su dominio. Los privilegios otorgados por esta opción pueden ser intimidantes para usuarios sin experiencia, pero los usuarios experimentados encontrarán la flexibilidad muy poderosa. También es posible, mediante modificaciones a la configuración de control de acceso predeterminada, permitir el acceso directo a la sección de dominios virtuales del archivo de configuración para Apache. Existen protecciones para prevenir el comportamiento malicioso, pero es poco probable que sea perfecto. Por defecto, los privilegios otorgados son generalmente seguros para su uso en un entorno de alojamiento compartido.
