Format d'adresse pour les hôtes virtuels Apache
Cette option détermine le format d'adresse que Virtualmin utilisera dans les nouveaux blocs Apache <Virtualhost>. Une étoile correspondra aux demandes qui parviennent à n'importe quelle adresse, tandis qu'une IP ne correspondra qu'aux demandes à cette adresse spécifique. Le comportement par défaut est déterminé en examinant la version Apache et les directives NameVirtualHost existantes, le cas échéant.
