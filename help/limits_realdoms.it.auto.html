Server non alias consentiti massimi
Questo campo imposta un limite al numero di sub-server reali non alias che questo proprietario del server può creare. Questo può essere impostato su un valore inferiore ai server virtuali massimi consentiti per consentire all'utente di creare molti server alias, ma limitare il numero di server reali (che consumano più spazio su disco e risorse di sistema).
