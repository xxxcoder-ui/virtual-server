Клонировать виртуальный сервер
Эта страница позволяет вам дублировать существующий виртуальный сервер с новым доменным именем. Любой веб-контент, записи DNS, почтовые ящики, почтовые псевдонимы, базы данных и другие параметры, связанные с исходным доменом, будут дублироваться, если это возможно. Тем не менее, веб-контент не будет изменен каким-либо образом, поэтому, если виртуальный сервер содержит веб-приложение, которое было настроено для использования баз данных или домашнего каталога старого сервера, его необходимо будет вручную перенастроить после клонирования.

При дублировании виртуального сервера верхнего уровня вы также должны ввести новое имя пользователя и пароль для владения новым доменом. Если у сервера есть частный IP-адрес, для клона будет автоматически выделен новый IP-адрес.

Процесс клонирования будет копировать любые базы данных, связанные с исходным доменом, под новыми именами и дублировать их содержимое. Однако он НЕ будет обновлять сценарии, установленные в исходном домене и настроенные для использования исходных баз данных - это необходимо сделать вручную.
