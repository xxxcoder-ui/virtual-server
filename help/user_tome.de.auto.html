Normalerweise an diesen Benutzer liefern
Wenn diese Option aktiviert ist, werden E-Mails an diesen Benutzer nach Durchlaufen einer Spam-Filterung an seine Mailbox gesendet. Normalerweise müssen Sie dies nicht überprüfen, da dies automatisch geschieht, wenn keine andere Weiterleitung oder automatische Antwort aktiviert ist. Wenn Sie jedoch eine Weiterleitungsadresse oder eine automatische Antwort einrichten und dennoch möchten, dass der Benutzer eine Kopie seiner E-Mail behält, sollte diese Option ausgewählt werden.
