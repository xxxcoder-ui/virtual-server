Maximal zulässige Alias-Server
In diesem Feld wird die Anzahl der virtuellen Alias-Server begrenzt, die dieser Serverbesitzer erstellen darf. Diese Begrenzung wird häufig in Verbindung mit den maximal zulässigen Nicht-Alias-Servern verwendet , um einem Benutzer die Berechtigung zum Erstellen einer höheren oder unbegrenzten Anzahl von Aliasen für seine Domänen zu erteilen, während die Anzahl der realen virtuellen Server weiterhin begrenzt wird.
