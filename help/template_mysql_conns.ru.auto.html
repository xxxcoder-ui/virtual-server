Максимальное количество подключений для владельцев доменов
Это поле управляет количеством одновременных подключений к MySQL, которое может выполнить учетная запись владельца виртуального сервера. Это включает в себя подключения из скриптов, установленных Virtualmin, таких как Wordpress или Drupal. По умолчанию разрешено неограниченное количество подключений, но поскольку каждое из них потребляет память в процессе сервера MySQL, вы можете ограничить общее количество подключений в загруженной системе.

Ограничение, установленное здесь, будет применяться только при создании нового виртуального сервера или при включении входа в MySQL для существующего сервера.
