Apache Webserver (для виртуального хоста)
Включите этот параметр, если учетная запись владельца домена должна иметь возможность редактировать конфигурацию Apache для своего домена. Привилегии, предоставляемые этой опцией, могут быть пугающими для неопытных пользователей, но опытные пользователи найдут гибкость очень мощной. Также возможно, путем изменения конфигурации управления доступом по умолчанию, разрешить прямой доступ к разделу виртуальных доменов файла конфигурации для Apache. Существуют средства защиты для предотвращения злонамеренного поведения, но оно вряд ли будет идеальным. По умолчанию предоставленные привилегии, как правило, безопасны для использования в среде общего хостинга.
