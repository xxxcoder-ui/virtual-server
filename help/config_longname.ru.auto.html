Стиль доменного имени в именах пользователей
Этот параметр определяет, будет ли доменное имя, используемое в именах пользователей, содержащих домен, полным доменным именем, то есть «virtualmin.com», только первой частью, то есть «virtualmin», или именем пользователя / группы для домена. По умолчанию имя группы используется для новых серверов верхнего уровня, а первая часть домена используется для подчиненных серверов.

В некоторых средах пользователям может быть проще понять имя пользователя, например "joeuser@virtualmin.com", для своего имени входа POP / IMAP или FTP, чем "joeuser.virtualmin".

Это поле также определяет способ вычисления префикса нового виртуального сервера. Варианты:

Полное доменное имя
Префикс всегда совпадает с именем домена.

Имя пользователя или первая часть домена
Префикс совпадает с именем группы Unix виртуального сервера, если это виртуальный сервер верхнего уровня. В противном случае применяется следующая опция.

Первая часть домена
Префикс является первой частью домена, если другой виртуальный сервер уже не использует этот префикс. Таким образом, для домена foo.com.au префикс будет (в порядке предпочтения) foo , foo.com или foo.com.au

Он также определяет автоматически сгенерированное имя пользователя для домена верхнего уровня, которое будет таким же, как префикс.
