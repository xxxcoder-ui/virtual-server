Standard-IPv6-Adresse für neue virtuelle Server verwenden?
Standardmäßig (wenn diese Option auf Nein gesetzt ist ) fügt Virtualmin neuen virtuellen Servern keine DNS-Einträge oder eine Webserverkonfiguration für die Standard-IPv6-Adresse des Systems hinzu. Wenn Sie hier jedoch Ja auswählen, wird die Standardeinstellung für neue Domänen geändert, um die primäre IPv6-Adresse des Systems zu verwenden. Zum Zeitpunkt der Erstellung kann der Benutzer weiterhin auswählen, ob eine private oder gar keine IPv6-Adresse verwendet werden soll.
