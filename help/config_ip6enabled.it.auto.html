Utilizzare l'indirizzo IPv6 predefinito per i nuovi server virtuali?
Per impostazione predefinita (quando questa opzione è impostata su No ), Virtualmin non aggiungerà voci DNS o una configurazione server Web a nuovi server virtuali per l'indirizzo IPv6 predefinito del sistema. Tuttavia, selezionando Sì qui verrà modificato il valore predefinito per i nuovi domini in modo da utilizzare l'indirizzo IPv6 primario del sistema. Al momento della creazione, l'utente può comunque selezionare di utilizzare un indirizzo IPv6 privato o nessuno.
