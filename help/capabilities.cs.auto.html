Může upravovat virtuální server
Umožňuje úpravy funkcí virtuálního serveru, jeho hesla a popisu.
Může spravovat uživatele
Umožňuje prohlížet, upravovat, vytvářet a mazat uživatele poštovní schránky, FTP a databáze.
Může spravovat aliasy
Umožňuje prohlížet, upravovat, vytvářet a mazat e-mailové aliasy.
Může spravovat databáze
Umožňuje přidávat další virtuální databáze MySQL a PostgreSQL na tento virtuální server nebo z něj mazat.
Může instalovat skripty
Umožňuje instalaci skriptů třetích stran na tento virtuální server.
Může změnit IP adresu
Pokud má tento server soukromou adresu IP, tato možnost umožňuje jeho změnu.
Může spravovat certifikáty SSL
Umožňuje nahrát nebo znovu vygenerovat certifikát SSL a klíč pro tento server.
Lze upravit rámec pro předávání
U webů určených pouze pro předávání umožňuje tato volba upravit cíl a HTML pro rámec pro předávání.
Může spravovat další administrátory
Umožňuje prohlížet, vytvářet, upravovat a mazat další účty správce vlastníka serveru.
Lze nakonfigurovat doručování spamu a virů
Umožňuje upravovat cíle a skóre doručení virů a skóre.
Může vybrat PHP verze
Pro domény s PHP, které běží přes CGI nebo fastCGI, umožňuje nastavení verzí PHP pro jednotlivé adresáře.
Může upravovat nastavení e-mailu
Umožňuje konfiguraci automatického BCCingu, režimu aliasu a dalších optoinů souvisejících s e-mailem.
Může vytvářet zálohy
Umožňuje zálohování virtuálních serverů na vzdálený server FTP nebo SSH.
Může naplánovat zálohování
Umožňuje pravidelné zálohování virtuálních serverů.
Lze obnovit zálohy
Umožňuje obnovit obsah domovského adresáře a databáze vlastněných virtuálních serverů.
Může vybrat sdílené adresy IP
Umožňuje vybrat globálně definované sdílené adresy IP při vytváření virtuálních serverů.
Může vytvářet aliasy
Umožňuje vytváření e-mailových aliasů, které předávají veškerou poštu do domény na jinou doménu nebo adresu.
Může použít editor webových stránek
Umožňuje úpravy webových stránek pomocí uživatelského rozhraní Virtualmin.
Může upravovat vzdálené počítače hostitele
Umožňuje vstup vzdálených klientských systémů, které mají povoleno připojení k databázím MySQL v doméně.
Může zakázat virtuální servery
Umožňuje dočasně deaktivovat servery.
Může odstranit virtuální servery
Umožňuje úplné odstranění vlastněných serverů, včetně všech dílčích serverů a aliasů.
