Administrador de archivos (solo directorio de inicio)
El Administrador de archivos es un applet Java avanzado que proporciona una vista navegable de doble panel del sistema de archivos en el servidor. Es una de las características más potentes de Webmin, y permite incluso a usuarios sin experiencia administrar archivos, directorios y permisos, editar archivos de texto, cargar y descargar archivos, y mucho más. Por defecto, el propietario del dominio solo podrá ver su propio directorio y subdirectorios.
