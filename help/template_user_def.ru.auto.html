Добавить пользователя Apache в группу Unix для новых серверов?
Этот параметр определяет, будет ли пользователь, с которым работает Apache, добавлен в новую группу виртуальных хостов. Когда включено, разрешения могут быть значительно ужесточены в домашних каталогах пользователей. Когда более узкие разрешения (например, 750) объединяются с SuExec, владельцы доменов не могут видеть каталоги и файлы других владельцев доменов. Этот вариант рекомендуется на любой платформе, которая поддерживает вторичные группы.
