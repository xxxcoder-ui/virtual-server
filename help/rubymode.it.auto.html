Modalità di esecuzione dello script Ruby
Questa opzione determina se gli script Ruby vengono eseguiti come utente del server Web Apache (in genere httpd o apache ) o l'utente Unix proprietario del server virtuale. Il primo è più veloce in quanto significa che non è necessario avviare ulteriori processi per eseguire il codice Ruby, ma meno sicuro in quanto gli script leggono e scrivono i file come utente diverso dal proprietario del dominio.
