Vollständiger Pfad zur Master-PHP-Konfigurationsdatei
Dieses Feld muss auf den vollständigen Pfad zur Datei php.ini auf Ihrem System eingestellt sein. Es wird nur verwendet, wenn Virtualmin PHP-Skripte als Eigentümer des virtuellen Servers über Suexec ausführt und zur Änderung pro Server in das Ausgangsverzeichnis jeder Domäne kopiert wird.
