È possibile selezionare tra indirizzi IP condivisi?
Quando questa opzione è impostata su Sì ed esistono più indirizzi IP condivisi sul sistema, il rivenditore sarà in grado di selezionarne uno durante la creazione di un nuovo server virtuale. Altrimenti sarà limitato agli IP condivisi selezionati nei campi sopra o se nessuno è scelto come indirizzo predefinito del sistema globale.
