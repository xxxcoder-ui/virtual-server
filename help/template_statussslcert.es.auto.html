Validar certificado SSL?
Esta opción determina si la validación del certificado SSL se habilitará para los nuevos servidores virtuales creados por Webmin que tienen SSL habilitado. Puede elegir que solo se verifique la fecha de vencimiento del certificado, o también validar que el nombre de dominio en el certificado coincida con la URL del servidor virtual.

Cuando el certificado de un dominio obtiene dentro de los 7 días posteriores a su vencimiento, Virtualmin enviará un correo electrónico de alerta al propietario del servidor virtual dándole tiempo para ordenar una renovación.
