Entregar a este usuario normalmente
Cuando esta opción está marcada, el correo electrónico a este usuario se enviará a su buzón, después de pasar por cualquier filtro de spam. Normalmente no necesita verificar esto, ya que sucederá automáticamente si no se habilita ningún otro reenvío o respuesta automática. Sin embargo, si configura una dirección de reenvío o una respuesta automática pero aún desea que el usuario conserve una copia de su correo electrónico, debe seleccionar esta opción.
