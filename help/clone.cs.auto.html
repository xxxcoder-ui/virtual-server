Klonovat virtuální server
Tato stránka umožňuje duplikovat existující virtuální server s novým názvem domény. Jakýkoli webový obsah, záznamy DNS, poštovní schránky, poštovní aliasy, databáze a další nastavení spojená s původní doménou budou duplikovány, pokud je to možné. Webový obsah se však nijak nezmění, takže pokud virtuální server obsahuje webovou aplikaci, která byla nakonfigurována pro použití databází nebo domovského adresáře starého serveru, bude nutné po klonování ručně znovu nakonfigurovat.

Při duplikování virtuálního serveru nejvyšší úrovně musíte také zadat nové uživatelské jméno a heslo k vlastnictví nové domény. Pokud má server soukromou IP adresu, bude klonu automaticky přidělena nová IP adresa.

Proces klonování zkopíruje všechny databáze spojené s původní doménou pod novými jmény a duplikuje jejich obsah. NEBUDE však aktualizovat žádné skripty nainstalované v původní doméně, které jsou nakonfigurovány pro použití původních databází - to musí být provedeno ručně.
