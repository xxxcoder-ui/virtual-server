Bereiche für die automatische IP-Zuweisung
Wenn virtuelle FTP-Hosts mit dieser Vorlage erstellt werden, können Sie einen Bereich von IP-Adressen angeben, aus denen zugewiesen werden soll. Virtuelle FTP-Hosts erfordern eine dedizierte IP-Adresse, da das Protokoll kein namensbasiertes virtuelles Hosting unterstützt.
