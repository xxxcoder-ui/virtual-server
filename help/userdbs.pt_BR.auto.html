Permitir acesso a bancos de dados
Este campo permite selecionar em quais bancos de dados pertencentes a este servidor virtual o usuário também pode efetuar login. Para adicionar um banco de dados, selecione-o na lista à esquerda e clique na seta ->.

O login e a senha do usuário serão os mesmos que ele usaria para FTP ou IMAP. Por padrão, ele terá acesso total a todas as tabelas nos bancos de dados selecionados.

O MySQL suporta apenas nomes de usuário com até 16 caracteres, portanto, para um usuário com um nome de conta longo, apenas os 16 primeiros caracteres são necessários ao efetuar login no banco de dados.
