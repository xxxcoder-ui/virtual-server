Memorizzare le password in chiaro?
Quando questa opzione è impostata su Sì , Virtualmin memorizzerà le password in chiaro per i proprietari di dominio e le cassette postali. Ciò consente loro di essere visualizzati nell'interfaccia utente, ma aggiunge un potenziale rischio per la sicurezza se il sistema è compromesso. Selezionando No , tutti i nuovi server virtuali creati da questo modello memorizzeranno invece solo password con hash unidirezionali.

Poiché le applicazioni Web devono archiviare le password del database in testo normale, la password del database per un utente verrà impostata su una nuova password casuale se Virtualmin è impostato per non memorizzare le password in chiaro. È possibile visualizzare questa password accedendo a Modifica database - <Password.
