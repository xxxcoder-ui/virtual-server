Peut choisir des noms de base de données?
Si cette restriction est définie sur Non , le propriétaire du serveur ne sera pas autorisé à sélectionner le nom de base de données par défaut lors de la création d'un sous-serveur sur lequel une fonction de base de données est activée. Au lieu de cela, il sera toujours par défaut à la première partie du nom de domaine.

De plus, il ne pourra pas supprimer la base de données par défaut pour un domaine existant, car cela lui permettrait de renommer efficacement la base de données par défaut après la création du domaine.
