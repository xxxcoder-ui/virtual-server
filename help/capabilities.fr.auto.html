Peut modifier le serveur virtuel
Permet de modifier les fonctionnalités du serveur virtuel, son mot de passe et sa description.
Peut gérer les utilisateurs
Permet aux utilisateurs de boîtes aux lettres, FTP et bases de données d'être affichés, modifiés, créés et supprimés.
Peut gérer les alias
Permet d'afficher, de modifier, de créer et de supprimer des alias de messagerie.
Peut gérer des bases de données
Permet à des bases de données MySQL et PostgreSQL supplémentaires d'être ajoutées à ce serveur virtuel ou supprimées de celui-ci.
Peut installer des scripts
Permet aux scripts tiers d'être installés sur ce serveur virtuel.
Peut changer l'adresse IP
Si ce serveur a une adresse IP privée, cette option permet de la modifier.
Peut gérer les certificats SSL
Permet au certificat SSL et à la clé de ce serveur d'être téléchargés ou régénérés.
Peut modifier le cadre de transfert
Pour les sites Web de transfert uniquement, cette option permet de modifier la destination et le code HTML du cadre de transfert.
Peut gérer des administrateurs supplémentaires
Permet d'afficher, de créer, de modifier et de supprimer des comptes d'administrateur de propriétaire de serveur supplémentaires.
Peut configurer la distribution de spam et de virus
Permet de modifier les destinations et les scores de distribution de spam et de virus.
Peut sélectionner des versions PHP
Pour les domaines avec PHP exécutés via CGI ou fastCGI, permet de définir des versions PHP par répertoire.
Peut modifier les paramètres de messagerie
Permet la configuration du BCCing automatique, du mode de copie d'alias et d'autres optoins liés aux e-mails.
Peut faire des sauvegardes
Permet aux serveurs virtuels possédés d'être sauvegardés sur un serveur FTP ou SSH distant.
Peut planifier des sauvegardes
Permet aux serveurs virtuels d'être sauvegardés régulièrement.
Peut restaurer des sauvegardes
Permet de restaurer le répertoire de base et le contenu de la base de données des serveurs virtuels possédés.
Peut sélectionner des adresses IP partagées
Permet de sélectionner des adresses IP partagées définies globalement lors de la création de serveurs virtuels.
Peut créer des alias de catchall
Permet la création d'alias de messagerie qui transfèrent tous les messages vers un domaine vers un domaine ou une adresse différent.
Peut utiliser l'éditeur de pages Web
Permet aux pages Web d'être modifiées via l'interface utilisateur Virtualmin.
Peut modifier les hôtes distants de la base de données
Permet la saisie de systèmes clients distants autorisés à se connecter aux bases de données MySQL du domaine.
Peut désactiver les serveurs virtuels
Permet aux serveurs d'être temporairement désactivés.
Peut supprimer des serveurs virtuels
Permet de supprimer complètement les serveurs possédés, y compris tous les sous-serveurs et alias.
