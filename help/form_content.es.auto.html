¿Crear página web inicial?
Para que Virtualmin cree un conjunto de páginas web básicas para este nuevo servidor virtual, seleccione la opción Sí, con el texto a continuación e ingrese HTML o texto que aparecerá en la página de índice en el cuadro a continuación. Esto se combinará con el estilo seleccionado en Estilo para el contenido inicial para producir las páginas HTML reales.
