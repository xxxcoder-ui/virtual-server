Répertoire personnel
Par défaut, le répertoire de base d'un utilisateur FTP de site Web est le même que le répertoire Web de base du domaine. Cependant, vous pouvez le remplacer par un sous-répertoire à la place, pour le limiter à accéder aux fichiers uniquement dans ce répertoire. Cela suppose que le serveur FTP a été configuré pour restreindre les clients à leurs répertoires personnels uniquement.
