Показать последний логин в списке пользователей?
Если для этого параметра установлено значение Да , время последнего входа в систему для пользователей почтовых ящиков отображается на странице «Редактирование пользователей» . Однако, поскольку сбор данных из журналов может быть медленным, вы можете выбрать «Нет» в системе с большим количеством пользователей.
