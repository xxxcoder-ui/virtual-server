File Manager (solo directory home)
Il File Manager è un'applet Java avanzata che fornisce una vista navigabile a doppio pannello del file system sul server. È una delle funzionalità più potenti di Webmin e consente anche agli utenti inesperti di gestire file, directory e autorizzazioni, modificare file di testo, caricare e scaricare file e molto altro. Per impostazione predefinita, il proprietario del dominio sarà in grado di vedere solo la propria directory e le sue sottodirectory.
