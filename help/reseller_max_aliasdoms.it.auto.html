Server alias massimo
Questo campo consente di imporre un limite separato al numero di server virtuali alias che il rivenditore può creare. È possibile impostare questo e il server alias massimo su Illimitato e impostare il numero massimo di server non alias su un numero inferiore. Ciò consentirebbe al rivenditore di creare tutti i domini alias che desidera, limitando al contempo il numero di server di livello superiore e sotto-server che richiedono più risorse.
