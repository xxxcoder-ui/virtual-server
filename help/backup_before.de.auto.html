Befehl, der vor dem Backup ausgeführt werden soll
Wenn festgelegt, wird der eingegebene Befehl oder das eingegebene Skript als root ausgeführt, bevor die Sicherung planmäßig ausgeführt wird. Dies kann nützlich sein, um austauschbare oder entfernte Dateisysteme zu mounten, in die das Backup geschrieben werden soll.

Wenn der Befehl fehlschlägt (mit einem Exit-Status ungleich Null), wird der Rest der Sicherung nicht fortgesetzt.
