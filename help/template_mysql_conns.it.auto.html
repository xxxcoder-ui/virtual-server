Numero massimo di connessioni per i proprietari di domini
Questo campo controlla il numero di connessioni simultanee a MySQL che l'account del proprietario di un server virtuale può effettuare. Ciò include le connessioni dagli script installati da Virtualmin, come Wordpress o Drupal. L'impostazione predefinita è consentire un numero illimitato di connessioni, ma poiché ciascuna consuma memoria all'interno del processo del server MySQL, potresti voler limitare le connessioni totali su un sistema occupato.

Il limite impostato qui verrà applicato solo quando viene creato un nuovo server virtuale o quando è abilitato un accesso MySQL per un server esistente.
