Puede editar servidor virtual
Permite editar las características del servidor virtual, su contraseña y descripción.
Puede administrar usuarios
Permite ver, editar, crear y eliminar usuarios de buzones, FTP y bases de datos.
Puede administrar alias
Permite ver, editar, crear y eliminar alias de correo electrónico.
Puede administrar bases de datos
Permite agregar o eliminar bases de datos MySQL y PostgreSQL adicionales a este servidor virtual.
Puede instalar scripts
Permite instalar scripts de terceros en este servidor virtual.
Puede cambiar la dirección IP
Si este servidor tiene una dirección IP privada, esta opción permite que se cambie.
Puede administrar certificados SSL
Permite que el certificado SSL y la clave para este servidor se carguen o se vuelvan a generar.
Puede editar el marco de reenvío
Para los sitios web de solo reenvío, esta opción permite editar el destino y el HTML del marco de reenvío.
Puede administrar administradores adicionales
Permite ver, crear, editar y eliminar cuentas adicionales de administrador del propietario del servidor.
Puede configurar la entrega de spam y virus
Permite editar los destinos y las puntuaciones de envío de spam y virus.
Puede seleccionar versiones de PHP
Para dominios con PHP ejecutados a través de CGI o fastCGI, permite configurar versiones de PHP por directorio.
Puede editar la configuración del correo electrónico
Permite la configuración de BCCing automático, modo de copia de alias y otras opciones relacionadas con el correo electrónico.
Puede hacer copias de seguridad
Permite realizar copias de seguridad de servidores virtuales propios en un servidor FTP o SSH remoto.
Puede programar copias de seguridad
Permite que los servidores virtuales sean respaldados en un horario regular.
Puede restaurar copias de seguridad
Permite restaurar el directorio de inicio y el contenido de la base de datos de servidores virtuales propios.
Puede seleccionar IP compartidas
Permite seleccionar direcciones IP compartidas definidas globalmente al crear servidores virtuales.
Puede crear alias generales
Permite la creación de alias de correo electrónico que reenvían todo el correo a un dominio a un dominio o dirección diferente.
Puede usar el editor de páginas web
Permite editar páginas web a través de la interfaz de usuario de Virtualmin.
Puede editar hosts remotos de bases de datos
Permite el ingreso de sistemas de clientes remotos que pueden conectarse a las bases de datos MySQL del dominio.
Puede deshabilitar servidores virtuales
Permite que los servidores se deshabiliten temporalmente.
Puede eliminar servidores virtuales
Permite que los servidores de propiedad se eliminen por completo, incluidos todos los sub-servidores y alias.
