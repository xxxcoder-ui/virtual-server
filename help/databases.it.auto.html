Modifica database
Questa pagina elenca tutti i database che sono attualmente di proprietà del server virtuale e consente la creazione di database di aggiunta fino a qualsiasi limite in vigore per l'utente che ha effettuato l'accesso a Virtualmin. È inoltre possibile eliminare un altro database selezionandoli dall'elenco e premendo il pulsante Elimina selezionati .

Per visualizzare le tabelle e i dati effettivi in un database, fare clic sul collegamento Gestisci accanto al nome del database nella tabella. Per visualizzare le dimensioni del database, fare clic sul suo nome.

Quando si utilizza Virtualmin Pro, sotto l'elenco del database vengono visualizzati due moduli aggiuntivi per modificare il login e la password per gli utenti MySQL e PostgreSQL utilizzati da questo server virtuale. Ciò può essere utile se si desidera che l'accesso al database sia diverso dall'account di accesso dell'amministrazione principale, forse per motivi di sicurezza. Questi dovrebbero essere cambiati con cura, poiché tutti gli script PHP (come quelli degli installatori di script) che hanno il nome utente o la password nei loro file di configurazione non verranno aggiornati.
