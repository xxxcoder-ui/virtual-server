Verzeichnisse, die von Sicherungen ausgeschlossen werden sollen
Aus dieser Vorlage erstellte virtuelle Server schließen standardmäßig alle hier eingegebenen Verzeichnisse von Sicherungen aus. Dies kann nützlich sein, wenn Sie wissen, dass auf allen Ihren Domains ähnliche Apps ausgeführt werden oder ähnliche Inhalte vorhanden sind. Ausgeschlossene Verzeichnisse können nach der Domänenerstellung noch bearbeitet werden.
