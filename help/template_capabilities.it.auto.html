Funzionalità di modifica predefinite
Queste caselle di controllo determinano quali funzionalità dei proprietari dei domini dei server virtuali possono modificare per impostazione predefinita. Una volta creato un server, questi possono essere modificati nella pagina Modifica limiti proprietario.

Se l'opzione Automatica è selezionata, i limiti sono determinati in base al fatto che il proprietario del server virtuale sia autorizzato a creare o meno sub-server (controllato dal campo Limite sul numero di server virtuali ). In tal caso, avrà accesso a tutte le funzionalità. Altrimenti, può solo gestire utenti, alias e modificare pagine Web.
