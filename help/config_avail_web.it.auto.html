Server web Apache (per host virtuale)
Abilitare questa opzione se l'account del proprietario del dominio deve avere la possibilità di modificare la configurazione di Apache per il suo dominio. I privilegi concessi da questa opzione possono intimidire gli utenti inesperti, ma gli utenti esperti troveranno la flessibilità molto potente. È anche possibile, attraverso modifiche alla configurazione predefinita del controllo di accesso, consentire l'accesso diretto alla sezione dei domini virtuali del file di configurazione per Apache. Sono in atto protezioni per prevenire comportamenti dannosi, ma è improbabile che sia perfetto. Per impostazione predefinita, i privilegi concessi sono generalmente sicuri per l'uso in un ambiente di hosting condiviso.
