Подпроцессы PHP FCGId
Когда сценарии PHP для этого домена выполняются через FCGId, число процессов PHP, установленных в этом поле, будет постоянно работать для обслуживания запросов. Вы можете увеличить это значение по умолчанию, равное 4, чтобы увеличить задержку PHP-скрипта, или уменьшить его, чтобы уменьшить использование памяти.

Если установить значение None , процессы PHP будут запускаться только по мере необходимости и очищаться после некоторого периода бездействия.
