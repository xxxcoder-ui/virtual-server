Modo de ejecución de script PHP
Esta opción determina si los scripts PHP se ejecutan como el usuario del servidor web Apache (generalmente httpd o apache ), o el usuario Unix que posee el servidor virtual. El primero es más rápido, ya que significa que no es necesario iniciar procesos adicionales para ejecutar código PHP, pero menos seguro ya que los scripts leen y escriben archivos como un usuario que no sea el propietario del dominio.

Si su sistema lo admite, la opción FCGId es la mejor, ya que ejecuta scripts PHP rápidamente y como propietario del dominio. De lo contrario, a menos que su sistema esté muy cargado o la seguridad entre los usuarios no sea un problema, se recomienda la opción de envoltorio CGI .
