Cucharones Amazon S3
Esta página enumera todos los buckets de Amazon S3 propiedad de cuentas S3 que Virtualmin está configurado para usar. Los cubos también se pueden crear y editar desde esta página, y se editan las reglas de control de acceso y ciclo de vida de eliminación automática que se aplican a los archivos dentro de ellas.

El proceso de copia de seguridad de Virtualmin creará automáticamente nuevos cubos según sea necesario, pero en la ubicación de almacenamiento S3 predeterminada. Crear un depósito desde esta página le permite seleccionar una ubicación de almacenamiento más cercana a su sistema Virtualmin.
