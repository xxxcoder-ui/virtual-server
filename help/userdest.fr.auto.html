Destinations de transfert d'e-mails
Ce tableau de champs détermine ce qui arrive au courrier électronique envoyé à cet utilisateur. Normalement, il serait simplement livré directement dans la boîte aux lettres de l'utilisateur, mais si des désignations sont définies ici, elles seront utilisées à la place. Cela peut être utilisé pour configurer un répondeur automatique si l'utilisateur est absent en vacances, ou pour transférer son e-mail à une adresse sur un autre serveur.

Chaque destination a un type (choisi dans le menu), qui peut être l'un des suivants:

  • Adresse électronique
    L'e-mail sera transmis à l'adresse entrée dans le champ adjacent.

  • Adresses dans le fichier
    L'e-mail sera transmis à toutes les adresses répertoriées dans le fichier texte dont le chemin est entré dans le champ adjacent.

  • Écrire dans un fichier
    Un e-mail à l'alias sera ajouté au fichier spécifié. Tous les en-têtes et le corps brut non décodé seront inclus.

  • Flux à programmer
    Le programme entré dans le champ adjacent sera exécuté et un e-mail à l'alias lui sera transmis en entrée.

  • Réponse automatique à partir du fichier
    Lorsque l'e-mail est reçu par l'utilisateur, un message de réponse automatique est renvoyé à l'expéditeur. Le texte de cette réponse automatique sera extrait du fichier entré dans le champ de texte adjacent.

  • Appliquer un fichier de filtre
    Les e-mails envoyés à l'utilisateur seront traités conformément aux règles du fichier de filtre spécifié, ce qui peut entraîner son transfert vers d'autres adresses ou fichiers en fonction du contenu réel du message.

  • Boîte aux lettres de l'utilisateur
    L'e-mail sera envoyé directement à la boîte aux lettres de l'utilisateur saisie dans le champ de texte adjacent. En règle générale, vous devez saisir le nom de l'utilisateur lui-même, afin qu'il reçoive une copie de l'e-mail ainsi que le transfert vers d'autres destinations. Cependant, vous devez entrer le nom d'utilisateur de connexion POP3 complet, comme joe.foo au lieu de simplement joe .

Si vous choisissez les adresses dans le fichier , la réponse automatique à partir du fichier ou Appliquer les types de destination du fichier de filtre , lorsque vous rééditez les liens des utilisateurs étiquetés Modifier .. apparaîtra à côté de ces destinations pour modifier le contenu du fichier spécifié.
