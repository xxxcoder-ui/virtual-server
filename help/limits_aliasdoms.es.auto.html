Máximo de servidores de alias permitidos
Este campo establece un límite en la cantidad de servidores virtuales de solo alias que este propietario del servidor puede crear. Este límite a menudo se usa junto con los servidores sin alias máximos permitidos para otorgar permisos a un usuario para crear un número mayor o ilimitado de alias para sus dominios, al tiempo que limita el número de servidores virtuales reales.
