File Manager (bare hjemmekatalog)
File Manager er et avansert Java-applet som gir en visbar visning med to sider av filsystemet pÃ¥ serveren. Det er en av de kraftigste funksjonene i Webmin, og lar selv uerfarne brukere administrere filer, kataloger og tillatelser, redigere tekstfiler, laste opp og laste ned filer og mye mer. Som standard vil domeneeieren bare kunne se sin egen katalog og underkataloger.
