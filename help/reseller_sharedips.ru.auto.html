Можно выбрать из общих IP-адресов?
Если для этого параметра установлено значение Да и в системе существует несколько общих IP-адресов, посредник сможет выбрать один из них при создании нового виртуального сервера. В противном случае он будет ограничен либо общими IP-адресами, выбранными в полях выше, либо, если ни один из них не будет выбран по умолчанию для глобального системного адреса.
