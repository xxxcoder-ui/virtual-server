Utiliser l'adresse IPv6 par défaut pour les nouveaux serveurs virtuels?
Par défaut (lorsque cette option est définie sur Non ), Virtualmin n'ajoutera pas d'entrées DNS ou d'une configuration de serveur Web aux nouveaux serveurs virtuels pour l'adresse IPv6 par défaut du système. Cependant, la sélection de Oui ici modifiera la valeur par défaut pour les nouveaux domaines afin d'utiliser l'adresse IPv6 principale du système. Au moment de la création, l'utilisateur peut toujours choisir d'utiliser une adresse IPv6 privée ou aucune.
