Usar endereço IPv6 padrão para novos servidores virtuais?
Por padrão (quando essa opção está definida como Não ), o Virtualmin não adiciona entradas DNS ou uma configuração de servidor da web a novos servidores virtuais para o endereço IPv6 padrão do sistema. No entanto, selecionar Sim aqui alterará o padrão para novos domínios para usar o endereço IPv6 principal do sistema. No momento da criação, o usuário ainda pode optar por usar um endereço IPv6 privado ou nenhum.
