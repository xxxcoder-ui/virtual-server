Можете выбрать имена баз данных?
Если для этого ограничения установлено значение Нет , владелец сервера не сможет выбрать имя базы данных по умолчанию при создании вспомогательного сервера, на котором включена функция базы данных. Вместо этого по умолчанию всегда будет указываться первая часть имени домена.

Кроме того, он не сможет удалить базу данных по умолчанию для любого существующего домена, поскольку это позволит ему эффективно переименовать базу данных по умолчанию после создания домена.
