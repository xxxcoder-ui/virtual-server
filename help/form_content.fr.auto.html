Créer la page Web initiale?
Pour que Virtualmin crée un ensemble de pages Web de base pour ce nouveau serveur virtuel, sélectionnez l'option Oui, avec le texte ci-dessous et entrez le HTML ou le texte qui apparaîtra sur la page d'index dans la case ci-dessous. Ceci sera combiné avec le style sélectionné dans Style pour le contenu initial afin de produire les pages HTML réelles.
