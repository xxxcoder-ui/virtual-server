Estilo de nombre de dominio en nombres de usuario
Esta opción determina si el nombre de dominio utilizado en los nombres de usuario que contienen el dominio será el nombre de dominio completo, es decir, "virtualmin.com", solo la primera parte, es decir, "virtualmin", o el nombre de usuario / grupo para el dominio. De manera predeterminada, el nombre del grupo se usa para los nuevos servidores de nivel superior, y la primera parte del dominio se usa para los sub-servidores.

En algunos entornos, puede ser más fácil para los usuarios comprender un nombre de usuario como "joeuser@virtualmin.com" para su nombre de inicio de sesión POP / IMAP o FTP que "joeuser.virtualmin".

Este campo también determina cómo se calcula el prefijo de un nuevo servidor virtual. Las opciones son:

Nombre de dominio completo
El prefijo es siempre el mismo que el nombre de dominio.

Nombre de usuario o primera parte del dominio
El prefijo es el mismo que el nombre del grupo Unix del servidor virtual, si se trata de un servidor virtual de nivel superior. De lo contrario, se aplica la siguiente opción.

Primera parte del dominio.
El prefijo es la primera parte del dominio, a menos que otro servidor virtual ya esté usando ese prefijo. Entonces, para el dominio foo.com.au , el prefijo sería (en orden de preferencia) foo , foo.com o foo.com.au

También determina el nombre de usuario generado automáticamente para un dominio de nivel superior, que será el mismo que el prefijo.
