Actie voor andere afzenders
Dit menu bepaald hoe ontvanger email servers de email afhandelen die van dit domein lijken te komen, maar niet afkomt van een toegestane server die gespecificeerd is in het veld hierboven. Indien u er absoluut zeker van bent dat alle email voor het domein gepasseerd is door deze virtuele server host of een andere server uit de toegestane lijst kunt u weigeren selecteren.
