Создать начальную веб-страницу?
Чтобы Virtualmin создал набор основных веб-страниц для этого нового виртуального сервера, выберите вариант «Да, с текстом ниже» и введите HTML или текст, который будет отображаться на странице индекса, в поле ниже. Это будет сочетаться со стилем, выбранным в разделе «Стиль» для исходного содержимого, для создания реальных страниц HTML.
