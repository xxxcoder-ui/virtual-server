Puede elegir nombres de bases de datos?
Si esta restricción se establece en No , el propietario del servidor no podrá seleccionar el nombre de la base de datos predeterminada al crear un subservidor que tenga habilitada una función de base de datos. En cambio, siempre se establecerá de manera predeterminada en la primera parte del nombre de dominio.

Además, no podrá eliminar la base de datos predeterminada para ningún dominio existente, ya que esto le permitiría cambiar el nombre de la base de datos predeterminada después de la creación del dominio.
