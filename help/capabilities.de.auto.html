Kann den virtuellen Server bearbeiten
Ermöglicht die Bearbeitung der Funktionen des virtuellen Servers, seines Kennworts und seiner Beschreibung.
Kann Benutzer verwalten
Ermöglicht das Anzeigen, Bearbeiten, Erstellen und Löschen von Postfach-, FTP- und Datenbankbenutzern.
Kann Aliase verwalten
Ermöglicht das Anzeigen, Bearbeiten, Erstellen und Löschen von E-Mail-Aliasen.
Kann Datenbanken verwalten
Ermöglicht das Hinzufügen oder Löschen zusätzlicher MySQL- und PostgreSQL-Datenbanken zu diesem virtuellen Server.
Kann Skripte installieren
Ermöglicht die Installation von Skripten von Drittanbietern auf diesem virtuellen Server.
Kann die IP-Adresse ändern
Wenn dieser Server eine private IP-Adresse hat, kann diese Option geändert werden.
Kann SSL-Zertifikate verwalten
Ermöglicht das Hochladen oder erneute Generieren des SSL-Zertifikats und des Schlüssels für diesen Server.
Kann den Weiterleitungsrahmen bearbeiten
Bei Nur-Weiterleitungs-Websites können mit dieser Option das Ziel und der HTML-Code für den Weiterleitungsrahmen bearbeitet werden.
Kann zusätzliche Administratoren verwalten
Ermöglicht das Anzeigen, Erstellen, Bearbeiten und Löschen zusätzlicher Administratorkonten für Serverbesitzer.
Kann die Zustellung von Spam und Viren konfigurieren
Ermöglicht die Bearbeitung von Spam- und Virenübermittlungszielen und -ergebnissen.
Kann PHP-Versionen auswählen
Ermöglicht für Domänen mit PHP, die über CGI oder fastCGI ausgeführt werden, das Festlegen von PHP-Versionen pro Verzeichnis.
Kann E-Mail-Einstellungen bearbeiten
Ermöglicht die Konfiguration von automatischem BCCing, Alias-Kopiermodus und anderen E-Mail-bezogenen Optionen.
Kann Backups erstellen
Ermöglicht das Sichern von virtuellen Servern auf einem Remote-FTP- oder SSH-Server.
Kann Backups planen
Ermöglicht die regelmäßige Sicherung virtueller Server.
Kann Backups wiederherstellen
Ermöglicht die Wiederherstellung des Ausgangsverzeichnisses und des Datenbankinhalts der eigenen virtuellen Server.
Kann gemeinsam genutzte IPs auswählen
Ermöglicht die Auswahl global definierter gemeinsam genutzter IP-Adressen beim Erstellen virtueller Server.
Kann Catchall-Aliase erstellen
Ermöglicht die Erstellung von E-Mail-Aliasen, die alle E-Mails an eine Domain an eine andere Domain oder Adresse weiterleiten.
Kann den Webseiten-Editor verwenden
Ermöglicht das Bearbeiten von Webseiten über die Virtualmin-Benutzeroberfläche.
Kann Datenbank-Remote-Hosts bearbeiten
Ermöglicht die Eingabe von Remote-Client-Systemen, die eine Verbindung zu den MySQL-Datenbanken der Domäne herstellen dürfen.
Kann virtuelle Server deaktivieren
Ermöglicht das vorübergehende Deaktivieren von Servern.
Kann virtuelle Server löschen
Ermöglicht das vollständige Löschen eigener Server, einschließlich aller Subserver und Aliase.
