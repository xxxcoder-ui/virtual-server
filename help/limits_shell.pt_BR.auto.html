Tipo de login permitido
Essa configuração determina como o administrador desse domínio pode fazer login, alterando seu shell Unix. As opções disponíveis padrão são:

Somente email
Os logins FTP e SSH estão bloqueados, mas o POP3 e o IMAP funcionarão.
Email e FTP
O usuário pode se conectar via FTP, mas não via SSH ou telnet.
Email, FTP e SSH
Logins via FTP, SSH e telnet são permitidos.
No entanto, eles podem ser personalizados pelo administrador principal na página Shell Personalizados .

Como esse recurso funciona alterando o shell do usuário do servidor virtual Unix, ele pressupõe que o servidor FTP use o arquivo /etc/shells para autenticação.
