Erzwingen Sie, dass sich virtuelle Server unter der Domäne befinden
Standardmäßig ist dies auf " Jede Domain zulassen" eingestellt, sodass der Reseller beliebige Domainnamen erstellen kann. Sie können ihn jedoch verwenden, um ihn auf das Erstellen unter einer bestimmten übergeordneten Domäne zu beschränken, normalerweise einer, die er besitzt.
