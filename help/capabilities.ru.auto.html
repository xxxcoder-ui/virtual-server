Можно редактировать виртуальный сервер
Позволяет редактировать характеристики виртуального сервера, его пароль и описание.
Может управлять пользователями
Позволяет просматривать, редактировать, создавать и удалять пользователей почтовых ящиков, FTP и баз данных.
Может управлять псевдонимами
Позволяет просматривать, редактировать, создавать и удалять почтовые псевдонимы.
Может управлять базами данных
Позволяет добавлять дополнительные базы данных MySQL и PostgreSQL к этому виртуальному серверу или удалять с него.
Можно установить скрипты
Позволяет устанавливать сторонние скрипты на этот виртуальный сервер.
Можно изменить IP-адрес
Если у этого сервера есть частный IP-адрес, этот параметр позволяет изменить его.
Может управлять SSL-сертификатами
Позволяет SSL-сертификат и ключ для этого сервера для загрузки или повторной генерации.
Можно редактировать кадр пересылки
Для веб-сайтов, предназначенных только для пересылки, этот параметр позволяет редактировать место назначения и HTML-код для рамки пересылки.
Может управлять дополнительными админами
Позволяет просматривать, создавать, редактировать и удалять дополнительные учетные записи администратора владельца сервера.
Можно настроить доставку спама и вирусов
Позволяет редактировать места назначения спама и вирусов и оценки.
Можно выбрать версии PHP
Для доменов с PHP, запускаемым через CGI или fastCGI, позволяет устанавливать версии PHP для каждого каталога.
Можно редактировать настройки электронной почты
Позволяет настроить автоматический BCCing, режим копирования псевдонимов и другие опции, связанные с электронной почтой.
Можно сделать резервные копии
Позволяет создавать резервные копии собственных виртуальных серверов на удаленном сервере FTP или SSH.
Можно запланировать резервное копирование
Позволяет создавать резервные копии виртуальных серверов на регулярной основе.
Можно восстановить резервные копии
Позволяет восстановить домашний каталог и содержимое базы данных виртуальных серверов.
Можно выбрать общие IP-адреса
Позволяет определять глобально определенные общие IP-адреса при создании виртуальных серверов.
Может создавать псевдонимы
Позволяет создавать псевдонимы электронной почты, которые перенаправляют всю почту в домен на другой домен или адрес.
Можно использовать редактор веб-страниц
Позволяет редактировать веб-страницы через интерфейс Virtualmin.
Может редактировать базу данных удаленных хостов
Позволяет входить в удаленные клиентские системы, которым разрешено подключаться к базам данных MySQL домена.
Можно отключить виртуальные серверы
Позволяет серверам временно отключаться.
Можно удалить виртуальные серверы
Позволяет полностью удалить собственные серверы, включая все субсерверы и псевдонимы.
