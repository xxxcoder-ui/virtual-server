Ограничить демонстрационный режим только для чтения?
Если для этого ограничения выбрано Да , владелец сервера не сможет вносить какие-либо изменения в систему или свои виртуальные серверы. Тем не менее, пользовательский интерфейс Virtualmin по-прежнему будет функционировать точно так же, как обычно. Это может быть полезно для создания демо-счета для показа потенциальным клиентам.
