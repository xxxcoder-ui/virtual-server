Maximal zulässige virtuelle Server
In diesem Feld wird die Anzahl der virtuellen Server eines beliebigen Typs begrenzt, den dieser Benutzer erstellen kann. Wenn die Option Kann nicht erstellen ausgewählt ist, darf er überhaupt keine Server hinzufügen. Der übergeordnete Server und alle Unterserver werden in die Zählung für dieses Limit einbezogen.
