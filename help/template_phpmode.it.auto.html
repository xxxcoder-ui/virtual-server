Modalità di uscita PHP predefinita
Questa opzione determina se gli script PHP nei nuovi server virtuali vengono eseguiti come utente del server Web Apache (in genere httpd o apache ) o l'utente Unix proprietario del server virtuale. Il primo è più veloce in quanto significa che non è necessario avviare ulteriori processi per eseguire il codice PHP, ma meno sicuro in quanto gli script leggono e scrivono i file come utente diverso dal proprietario del dominio.

Se il tuo sistema lo supporta, l'opzione FCGId è la migliore, poiché esegue script PHP rapidamente e come proprietario del dominio. Altrimenti, a meno che il sistema non sia pesantemente caricato o la sicurezza tra gli utenti non sia un problema, si consiglia l'opzione wrapper CGI .
