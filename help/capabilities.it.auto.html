Può modificare il server virtuale
Consente la modifica delle funzionalità del server virtuale, della sua password e descrizione.
Può gestire gli utenti
Consente agli utenti di cassette postali, FTP e database di essere visualizzati, modificati, creati ed eliminati.
Può gestire gli alias
Consente di visualizzare, modificare, creare ed eliminare gli alias email.
Può gestire database
Consente di aggiungere database MySQL e PostgreSQL aggiuntivi a questo server virtuale o eliminarli da esso.
Può installare script
Consente l'installazione di script di terze parti in questo server virtuale.
Può cambiare l'indirizzo IP
Se questo server ha un indirizzo IP privato, questa opzione consente di modificarlo.
Può gestire certificati SSL
Consente il caricamento o la rigenerazione del certificato SSL e della chiave per questo server.
Può modificare il frame di inoltro
Per i siti Web di solo inoltro, questa opzione consente di modificare la destinazione e l'HTML del frame di inoltro.
Può gestire amministratori aggiuntivi
Consente di visualizzare, creare, modificare ed eliminare gli account amministratore del proprietario del server extra.
Può configurare la consegna di spam e virus
Consente di modificare le destinazioni e i punteggi di consegna di spam e virus.
Può selezionare le versioni di PHP
Per i domini con PHP eseguiti tramite CGI o fastCGI, consente di impostare le versioni PHP per directory.
Può modificare le impostazioni e-mail
Consente la configurazione di BCCing automatici, la modalità di copia alias e altre opzioni opzionali relative alla posta elettronica.
Può fare backup
Consente il backup dei server virtuali di proprietà su un server FTP o SSH remoto.
Può pianificare i backup
Consente il backup dei server virtuali con una pianificazione regolare.
Può ripristinare i backup
Consente il ripristino della home directory e del contenuto del database dei server virtuali di proprietà.
Può selezionare IP condivisi
Consente di selezionare gli indirizzi IP condivisi definiti a livello globale durante la creazione di server virtuali.
Può creare alias catchall
Consente la creazione di alias e-mail che inoltrano tutta la posta a un dominio a un dominio o indirizzo diverso.
Può usare l'editor di pagine Web
Consente di modificare le pagine Web tramite l'interfaccia utente di Virtualmin.
Può modificare gli host remoti del database
Consente l'immissione di sistemi client remoti a cui è consentito connettersi ai database MySQL del dominio.
Può disabilitare i server virtuali
Consente ai server di essere temporaneamente disabilitati.
Può eliminare i server virtuali
Consente di eliminare completamente i server di proprietà, inclusi tutti i server secondari e gli alias.
