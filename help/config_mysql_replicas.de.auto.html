Zusätzliche MySQL-Server zum Erstellen von Benutzern
Standardmäßig erstellt Virtualmin alle MySQL-Konten und -Berechtigungen auf einem einzelnen MySQL-Server, der normalerweise auf demselben Computer wie Virtualmin selbst ausgeführt wird. Wenn Sie jedoch mehrere Server mit Replikation oder dem NDBCluster-System ausführen, benötigen Sie möglicherweise Virtualmin, um diese Berechtigungen auf mehreren Servern zu erstellen. Auf diese Weise können sich Domaininhaber bei einem von ihnen anmelden.

Geben Sie zum Einrichten eine durch Leerzeichen getrennte Liste zusätzlicher Server in dieses Feld ein. Von nun an erstellte virtuelle Server werden dem Mastersystem und den in der Liste aufgeführten hinzugefügt. Vorhandene virtuelle Server werden jedoch nicht hinzugefügt.
