BIND DNS Server (für DNS Domain)
Aktivieren Sie diese Option, wenn das Domänenbesitzer-Konto DNS-Einträge für seine Domäne bearbeiten soll. Abhängig von der Konfiguration kann der Domänenbesitzer neue Unterdomäneneinträge erstellen, vorhandene Domäneneinträge ändern, MX-Einträge ändern und vieles mehr. Unerfahrene Benutzer möchten oder benötigen möglicherweise nicht so viel Flexibilität.
