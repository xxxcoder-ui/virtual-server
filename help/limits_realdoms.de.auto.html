Maximal zulässige Nicht-Alias-Server
In diesem Feld wird die Anzahl der realen Nicht-Alias-Subserver begrenzt, die dieser Serverbesitzer erstellen darf. Dies kann niedriger als die maximal zulässigen virtuellen Server festgelegt werden , damit der Benutzer viele Alias-Server erstellen kann, aber die Anzahl der realen Server (die mehr Speicherplatz und Systemressourcen verbrauchen) einschränken kann.
