Mag database namen kiezen?
Als deze beperking is ingesteld op Nee is het niet mogelijk voor de server eigenaar om de standaard database naam te kiezen wanneer hij een sub-server aanmaakt waarbij de database functie aanstaat. In plaats hiervan zal standaard altijd het eerste gedeelte van de domein naam gebruikt worden.

Daarnaast kan hij niet de standaard database verwijderen van een bestaand domein, als dit wel zou kunnen zou hij ook de de standaard database kunnen hernoemen nadat het domein was aangemaakt.
