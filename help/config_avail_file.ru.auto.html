Файловый менеджер (только домашний каталог)
Диспетчер файлов - это продвинутый Java-апплет, который обеспечивает просмотр файловой системы на сервере с двойным панорамированием. Это одна из самых мощных функций Webmin, которая позволяет даже неопытным пользователям управлять файлами, каталогами и разрешениями, редактировать текстовые файлы, загружать и скачивать файлы и многое другое. По умолчанию владелец домена сможет видеть только свой каталог и подкаталоги.
