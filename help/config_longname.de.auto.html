Domain Name Style in Benutzernamen
Diese Option bestimmt, ob der Domänenname, der in Benutzernamen verwendet wird, die die Domäne enthalten, der vollständige Domänenname ist, dh "virtualmin.com", nur der erste Teil, dh "virtualmin", oder der Benutzer- / Gruppenname für die Domäne. Standardmäßig wird der Gruppenname für neue Server der obersten Ebene verwendet, und der erste Teil der Domäne wird für Unterserver verwendet.

In einigen Umgebungen ist es für Benutzer möglicherweise einfacher, einen Benutzernamen wie "joeuser@virtualmin.com" für ihren POP / IMAP- oder FTP-Anmeldenamen zu verstehen als "joeuser.virtualmin".

Dieses Feld bestimmt auch, wie das Präfix eines neuen virtuellen Servers berechnet wird. Die Optionen sind:

Vollständiger Domainname
Der Präfix ist immer der gleiche wie der Domainname.

Benutzername oder erster Teil der Domain
Das Präfix entspricht dem Unix-Gruppennamen des virtuellen Servers, wenn es sich um einen virtuellen Server der obersten Ebene handelt. Andernfalls gilt die folgende Option.

Erster Teil der Domain
Das Präfix ist der erste Teil der Domäne, es sei denn, ein anderer virtueller Server verwendet dieses Präfix bereits. Für die Domain foo.com.au wäre das Präfix (in der Reihenfolge der Präferenz) foo , foo.com oder foo.com.au

Außerdem wird der automatisch generierte Benutzername für eine Top-Level-Domain ermittelt, der mit dem Präfix identisch ist.
