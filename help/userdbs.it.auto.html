Consentire l'accesso ai database
Questo campo consente di selezionare i database di proprietà di questo server virtuale a cui l'utente può anche accedere. Per aggiungere un database, selezionarlo dall'elenco a sinistra e fare clic sulla freccia ->.

Il login e la password dell'utente saranno gli stessi di quelli che userebbe per FTP o IMAP. Per impostazione predefinita, gli verrà concesso il pieno accesso a tutte le tabelle nei database selezionati.

MySQL supporta solo nomi utente lunghi fino a 16 caratteri, quindi per un utente con un nome account lungo sono necessari solo i primi 16 caratteri quando si accede al database.
