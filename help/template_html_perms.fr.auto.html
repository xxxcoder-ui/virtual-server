Autorisations sur le répertoire du site Web
Cette option spécifie les autorisations initiales sur les fichiers et répertoires du répertoire du site Web spécifiés dans l'option précédente. Si SuExec a été activé et que l'utilisateur Apache est ajouté au groupe d'utilisateurs du serveur virtuel, ces autorisations peuvent être renforcées pour empêcher les autres utilisateurs de voir le contenu du répertoire du site Web. Cet ensemble d'autorisations resserré est recommandé pour tout déploiement, sauf si tous les utilisateurs disposant d'un accès shell sont des utilisateurs approuvés. Les autorisations recommandées sont 750, ce qui équivaut à rwxr-x ---. En d'autres termes, le propriétaire peut lire, écrire et exécuter, le groupe peut lire et exécuter et d'autres n'ont aucune autorisation pour lire, écrire ou exécuter des fichiers dans le répertoire.
