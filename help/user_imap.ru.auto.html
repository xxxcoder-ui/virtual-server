IMAP / FTP вход
В этом поле отображается имя пользователя, которое этот почтовый ящик будет использовать при входе в систему через IMAP для получения электронной почты, SMTP для отправки почты или FTP для управления файлами. Он отличается от первой части его адреса электронной почты, поскольку к нему добавлено или частично добавлено или добавлено доменное имя.
