Formato dell'indirizzo per host virtuali Apache
Questa opzione determina il formato dell'indirizzo che Virtualmin utilizzerà nei nuovi blocchi Apache <host virtuale>. Una stella abbinerà le richieste che arrivano a qualsiasi indirizzo, mentre un IP abbinerà solo le richieste a quell'indirizzo specifico. Il comportamento predefinito viene determinato osservando la versione di Apache e le eventuali direttive NameVirtualHost esistenti.
