Хранить незашифрованные пароли?
Если для этого параметра установлено значение Да , Virtualmin будет хранить пароли в виде открытого текста для владельцев доменов и почтовых ящиков. Это позволяет отображать их в пользовательском интерфейсе, но добавляет потенциальную угрозу безопасности, если ваша система взломана. Выбор Нет приведет к тому, что все новые виртуальные серверы, созданные из этого шаблона, будут хранить только односторонние хешированные пароли.

Поскольку веб-приложения должны хранить пароли базы данных в виде простого текста, пароль базы данных для пользователя будет установлен на новый случайный пароль, если Virtualmin настроен не хранить пароли в виде открытого текста. Вы можете просмотреть этот пароль, зайдя в Редактировать базы данных - <Пароли.
