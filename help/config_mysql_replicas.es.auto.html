Servidores MySQL adicionales para crear usuarios en
De manera predeterminada, Virtualmin creará todas las cuentas y permisos de MySQL en un solo servidor MySQL, generalmente el que se ejecuta en la misma máquina que Virtualmin. Sin embargo, si está ejecutando múltiples servidores con replicación o el sistema NDBCluster, puede necesitar Virtualmin para crear estos permisos en varios servidores. Esto permitirá a los propietarios de dominios iniciar sesión en cualquiera de ellos.

Para configurar esto, ingrese una lista separada por espacios de servidores adicionales en este campo. Los servidores virtuales creados a partir de ahora se agregarán al sistema maestro y a los de la lista. Sin embargo, los servidores virtuales existentes no se agregarán.
