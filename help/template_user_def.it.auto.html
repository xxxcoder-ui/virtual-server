Aggiungi l'utente Apache al gruppo Unix per i nuovi server?
Questa opzione determina se l'utente che Apache esegue come verrà aggiunto al nuovo gruppo host virtuale. Se abilitati, le autorizzazioni possono essere notevolmente rafforzate nelle home directory degli utenti. Quando permessi più stretti (cioè 750) vengono combinati con SuExec, è impossibile per i proprietari di domini vedere directory e file di altri proprietari di domini. Questa opzione è consigliata su qualsiasi piattaforma che supporti gruppi secondari.
