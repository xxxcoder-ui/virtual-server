Sous-processus PHP FCGId
Lorsque les scripts PHP pour ce domaine sont exécutés via FCGId, le nombre de processus PHP définis dans ce champ sera toujours exécuté pour répondre aux demandes. Vous pouvez l'augmenter à partir de la valeur par défaut de 4 pour améliorer la latence des scripts PHP, ou la diminuer pour réduire l'utilisation de la mémoire.

Si vous le définissez sur None , les processus PHP ne seront lancés que si nécessaire à la demande et nettoyés après une certaine période d'inactivité.
