Répertoires à exclure des sauvegardes
Les serveurs virtuels créés à partir de ce modèle excluront par défaut tous les répertoires entrés ici des sauvegardes. Cela peut être utile si vous savez que tous vos domaines exécuteront des applications similaires ou auront un contenu similaire. Les répertoires exclus peuvent toujours être modifiés après la création du domaine.
