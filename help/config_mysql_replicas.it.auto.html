Server MySQL aggiuntivi su cui creare utenti
Per impostazione predefinita, Virtualmin creerà tutti gli account e le autorizzazioni MySQL su un singolo server MySQL, in genere quello in esecuzione sullo stesso computer di Virtualmin stesso. Tuttavia, se si eseguono più server con replica o il sistema NDBCluster, potrebbe essere necessario Virtualmin per creare queste autorizzazioni su più server. Ciò consentirà ai proprietari di domini di accedere a uno di essi.

Per configurarlo, inserisci un elenco separato da spazi di server aggiuntivi in questo campo. I server virtuali creati da ora in poi verranno aggiunti al sistema principale e a quelli nell'elenco. Tuttavia, i server virtuali esistenti non verranno aggiunti.
