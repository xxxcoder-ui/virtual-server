Editar bancos de dados
Esta página lista todos os bancos de dados que pertencem atualmente ao servidor virtual e permite a criação de bancos de dados adicionais até qualquer limite em vigor para o usuário conectado ao Virtualmin. Você também pode excluir mais um banco de dados selecionando-os na lista e pressionando o botão Excluir selecionados .

Para visualizar as tabelas e os dados reais em um banco de dados, clique no link Gerenciar ao lado do nome do banco de dados na tabela. Para ver o tamanho do banco de dados, clique no nome dele.

Ao usar o Virtualmin Pro, dois formulários adicionais são mostrados abaixo da lista do banco de dados para alterar o login e a senha dos usuários MySQL e PostgreSQL usados por este servidor virtual. Isso pode ser útil se você desejar que o logon do banco de dados seja diferente do logon da administração principal, talvez por motivos de segurança. Eles devem ser alterados com cuidado, pois os scripts PHP (como os dos instaladores de scripts) que possuem o nome de usuário ou a senha em seus arquivos de configuração não serão atualizados.
