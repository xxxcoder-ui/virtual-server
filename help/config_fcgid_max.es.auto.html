Tiempo máximo de ejecución de fcgid
De manera predeterminada, Virtualmin establece el tiempo de ejecución máximo para que los procesos fastCGI coincidan con el tiempo de ejecución de los scripts PHP, según lo determine el archivo php.ini del dominio. Esto asegura que las aplicaciones PHP no se ejecutarán durante más tiempo del que se supone que deben.

Sin embargo, puede deshabilitar este comportamiento seleccionando No establecer en este campo de configuración. O puede especificar su propio tiempo de espera para scripts fastCGI seleccionando Tiempo en segundos e ingresando un número.

Esto se implementa configurando la directiva Apache IPCCommTimeout , o el equivalente en Nginx.
