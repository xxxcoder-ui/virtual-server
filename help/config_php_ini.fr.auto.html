Chemin d'accès complet au fichier de configuration PHP maître
Ce champ doit être défini sur le chemin d'accès complet au fichier php.ini sur votre système. Il n'est utilisé que lorsque Virtualmin exécute des scripts PHP en tant que propriétaire du serveur virtuel via Suexec et est copié dans le répertoire de base de chaque domaine pour une modification par serveur.
