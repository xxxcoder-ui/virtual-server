Serveurs MySQL supplémentaires pour créer des utilisateurs sur
Par défaut, Virtualmin créera tous les comptes et autorisations MySQL sur un seul serveur MySQL, généralement celui qui s'exécute sur la même machine que Virtualmin lui-même. Cependant, si vous exécutez plusieurs serveurs avec réplication ou le système NDBCluster, vous devrez peut-être Virtualmin pour créer ces autorisations sur plusieurs serveurs. Cela permettra aux propriétaires de domaine de se connecter à l'un d'eux.

Pour configurer cela, entrez une liste séparée par des espaces de serveurs supplémentaires dans ce champ. Les serveurs virtuels créés à partir de maintenant seront ajoutés au système maître et à ceux de la liste. Les serveurs virtuels existants ne seront cependant pas ajoutés.
