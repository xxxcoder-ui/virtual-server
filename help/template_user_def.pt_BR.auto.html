Adicionar usuário Apache ao grupo Unix para novos servidores?
Esta opção determina se o usuário que o Apache executa como será adicionado ao novo grupo de hosts virtuais. Quando ativada, as permissões podem ser aumentadas significativamente nos diretórios pessoais do usuário. Quando permissões mais restritas (ou seja, 750) são combinadas com o SuExec, torna impossível para os proprietários de domínio visualizarem outros diretórios e arquivos de proprietários de domínio. Esta opção é recomendada em qualquer plataforma que suporte grupos secundários.
