Servidores de Correio Secundário
Um servidor de email secundário é aquele que também pode receber emails de um dos servidores virtuais gerenciados pelo Virtualmin, mas apenas como uma retransmissão temporária, e não como o destino final. É aconselhável configurar um servidor secundário para domínios de email muito usados, para atuar como um local de backup ou retenção, caso o servidor de email principal esteja inoperante.

Nesta página, você pode selecionar outros servidores com o Virtualmin instalado para ser usado como servidores MX secundários para novos domínios de correio criados neste sistema. Antes que os servidores apareçam, eles devem ser adicionados ao módulo Índice de servidores Webmin, no qual você deve inserir o login e a senha raiz do servidor remoto.

Para cada servidor de correio secundário, você tem a opção de inserir um nome de host diferente para usar no registro DNS MX, em vez do nome de host real do servidor, como secmx.seudominio.com . No entanto, verifique se o que você digitou é um nome DNS válido que é resolvido no mesmo servidor.

Se você clicar no botão Salvar e adicionar , todos os servidores virtuais existentes para os quais o email estiver ativado serão adicionados a todos os servidores de email secundários selecionados. Além disso, os registros MX DNS serão adicionados aos seus domínios para se referir aos servidores de correio.
