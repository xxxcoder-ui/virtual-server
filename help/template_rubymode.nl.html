Standaard Ruby uitvoer mode
Deze optie bepaald of de Ruby scripts in een nieuwe virtuele server worden uitgevoerd als de Apache web server gebruiker (normaal httpd of apache), of als de Unix gebruiker waar de virtuele server van is. De eerste is sneller omdat het geen extra processen hoeft op te starten om de Ruby code uit te voeren, echter is minder veilig omdat scripts de files lezen en schrijven als een gebruiker en niet als een domein eigenaar.
