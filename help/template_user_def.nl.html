Apache gebruiker toevoegen aan Unix groep voor nieuwe servers?
Deze optie bepaald of de gebruiker waar Apache werkt word toegevoegd aan de nieuwe virtuele host groep. Wanneer dit is aangezet kunnen permissies strenger gemaakt worden op de gebruikers home directory's. Wanneer de strengere permissies worden (bijv. 750) worden gecombineerd met SuExec, is het onmogelijk voor domein eigenaren om andere domein eigenaar directory's en files te bekijken. Deze optie word aanbevolen op ieder platform die secundaire groepen ondersteund.
