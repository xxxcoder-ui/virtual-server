Servidor Web Apache (para host virtual)
Habilite isso se a conta do proprietário do domínio tiver a capacidade de editar a configuração do Apache para seu domínio. Os privilégios concedidos por essa opção podem ser intimidantes para usuários inexperientes, mas usuários experientes encontrarão a flexibilidade muito poderosa. Também é possível, através de modificações na configuração padrão de controle de acesso, permitir acesso direto à seção de domínios virtuais do arquivo de configuração do Apache. Existem proteções para evitar comportamentos maliciosos, mas é improvável que seja perfeito. Por padrão, os privilégios concedidos geralmente são seguros para uso em um ambiente de hospedagem compartilhada.
