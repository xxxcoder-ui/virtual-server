Apache-Benutzer zur Unix-Gruppe für neue Server hinzufügen?
Diese Option bestimmt, ob der Benutzer, unter dem Apache ausgeführt wird, der neuen virtuellen Hostgruppe hinzugefügt wird. Wenn diese Option aktiviert ist, können die Berechtigungen für Benutzerverzeichnisse erheblich eingeschränkt werden. Wenn strengere Berechtigungen (dh 750) mit SuExec kombiniert werden, können Domaininhaber keine Verzeichnisse und Dateien anderer Domaininhaber anzeigen. Diese Option wird auf jeder Plattform empfohlen, die sekundäre Gruppen unterstützt.
