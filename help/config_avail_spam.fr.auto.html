SpamAssassin Mail Filter (pour le fichier de configuration des domaines)
S'il est activé, le compte propriétaire du domaine aura accès au module de configuration SpamAssassin pour son domaine. Avec ce module, les politiques de SpamAssassin peuvent être personnalisées. Les utilisateurs inexpérimentés peuvent être intimidés par ce niveau de contrôle du filtre anti-spam et peuvent potentiellement entraîner une augmentation des faux positifs ou réduire accidentellement l'efficacité du filtre anti-spam.
