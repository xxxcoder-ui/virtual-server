Durée d'exécution maximale du script PHP
Lorsque les scripts PHP sont exécutés en mode FCGId, ce champ définit la durée maximale pendant laquelle chaque appel de script peut s'exécuter. Vous devrez peut-être l'augmenter lors de l'hébergement de scripts qui ont des temps d'exécution particulièrement élevés et qui sont tués par votre serveur Web avant la fin.
