Диапазоны для автоматического распределения IP
Если виртуальные хосты FTP будут создаваться с использованием этого шаблона, вы можете указать диапазон IP-адресов для выделения. Для виртуальных хостов FTP требуется выделенный IP-адрес, поскольку протокол не поддерживает виртуальный хостинг на основе имен.
