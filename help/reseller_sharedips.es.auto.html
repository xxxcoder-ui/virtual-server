¿Se puede seleccionar de direcciones IP compartidas?
Cuando esta opción se establece en Sí y existen varias direcciones IP compartidas en el sistema, el revendedor podrá seleccionar una al crear un nuevo servidor virtual. De lo contrario, se limitará a las IP compartidas seleccionadas en los campos anteriores o si no se elige ninguna a la dirección predeterminada del sistema global.
