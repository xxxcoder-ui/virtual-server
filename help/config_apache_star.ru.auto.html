Формат адреса для виртуальных хостов Apache
Эта опция определяет формат адреса, который Virtualmin будет использовать в новых блоках Apache <Virtualhost>. Звезда будет сопоставлять запросы, поступающие на любой адрес, а IP-адрес будет сопоставлять запросы только с этим конкретным адресом. Поведение по умолчанию определяется путем просмотра версии Apache и того, какие директивы NameVirtualHost существуют, если таковые имеются.
