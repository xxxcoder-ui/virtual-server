Kloon Virtuele Server
Met deze pagina kunt u een virtuele server dupliceren met een nieuwe domein naam. Alle web inhoud, DNS record's, mailboxen, email aliassen, databases en andere instellingen die horen bij het originele domein zullen worden gedupliceerd, indien mogelijk. Echter de webinhoud zal niet worden gewijzigd, dus indien de virtuele server een web applicatie heeft dat is geconfigureerd om te gebruiken op de oude server database of home directory kan het nodig zijn dat na het klonen een aantal dingen handmatig opnieuw geconfigureerd moeten worden.

Wanneer u een top-niveau virtuele server kloont moet u ook een nieuwe gebruikersnaam en wachtwoord invullen om het nieuwe domein in bezit te krijgen. Indien de server een prive IP adres heeft zal er automatisch een nieuw IP adres worden toegewezen voor de kloon.
