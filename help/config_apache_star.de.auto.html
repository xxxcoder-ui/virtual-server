Adressformat für virtuelle Apache-Hosts
Diese Option bestimmt das Adressformat, das Virtualmin in neuen Apache <Virtualhost> -Blöcken verwendet. Ein Stern stimmt mit Anfragen überein, die an eine beliebige Adresse kommen, während eine IP nur Anfragen mit dieser bestimmten Adresse abgleichen kann. Das Standardverhalten wird anhand der Apache-Version und der vorhandenen NameVirtualHost- Anweisungen bestimmt.
