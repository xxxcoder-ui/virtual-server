Creare una pagina Web iniziale?
Per fare in modo che Virtualmin crei un set di pagine Web di base per questo nuovo server virtuale, selezionare l'opzione Sì, con il testo sottostante e immettere HTML o testo che verrà visualizzato nella pagina dell'indice nel riquadro in basso. Questo sarà combinato con lo stile selezionato in Stile per il contenuto iniziale per produrre le pagine HTML effettive.
