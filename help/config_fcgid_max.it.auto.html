Tempo di esecuzione massimo di fcgid
Per impostazione predefinita, Virtualmin imposta il tempo di esecuzione massimo per i processi fastCGI in modo che corrisponda al tempo di esecuzione degli script PHP, come determinato dal file php.ini del dominio. Questo garantisce che le applicazioni PHP non funzioneranno più a lungo di quanto dovrebbero.

Tuttavia, è possibile disabilitare questo comportamento selezionando Non impostare in questo campo di configurazione. Oppure puoi specificare il tuo timeout per gli script fastCGI selezionando Tempo in secondi e inserendo un numero.

Questo viene implementato impostando la direttiva IPCCommTimeout o l'equivalente in Nginx.
