Полный путь к главному файлу конфигурации PHP
В этом поле должен быть указан полный путь к файлу php.ini в вашей системе. Он используется только тогда, когда Virtualmin запускает сценарии PHP как владелец виртуального сервера через Suexec, и копируется в домашний каталог каждого домена для модификации на сервере.
