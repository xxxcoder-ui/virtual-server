Estilo de nome de domínio em nomes de usuário
Esta opção determina se o nome de domínio usado nos nomes de usuário que contêm o domínio será o nome completo, como "virtualmin.com", apenas a primeira parte, como "virtualmin" ou o nome de usuário / grupo do domínio. Por padrão, o nome do grupo é usado para novos servidores de nível superior e a primeira parte do domínio é usada para sub-servidores.

Em alguns ambientes, pode ser mais fácil para os usuários entenderem um nome de usuário como "joeuser@virtualmin.com" para o nome de login POP / IMAP ou FTP do que "joeuser.virtualmin".

Este campo também determina como o prefixo de um novo servidor virtual é calculado. As opções são:

Nome de domínio completo
O prefx é sempre o mesmo que o nome do domínio.

Nome de usuário ou primeira parte do domínio
O prefixo é o mesmo que o nome do grupo Unix do servidor virtual, se for um servidor virtual de nível superior. Caso contrário, a opção a seguir se aplica.

Primeira parte do domínio
O prefixo é a primeira parte do domínio, a menos que outro servidor virtual já esteja usando esse prefixo. Portanto, para o domínio foo.com.au , o prefixo seria (em ordem de preferência) foo , foo.com.au ou foo.com

Ele também determina o nome de usuário gerado automaticamente para um domínio de nível superior, que será o mesmo que o prefixo.
