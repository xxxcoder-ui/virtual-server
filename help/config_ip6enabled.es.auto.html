¿Usar la dirección IPv6 predeterminada para nuevos servidores virtuales?
De forma predeterminada (cuando esta opción se establece en No ), Virtualmin no agregará entradas DNS o una configuración de servidor web a los nuevos servidores virtuales para la dirección IPv6 predeterminada del sistema. Sin embargo, si selecciona Sí aquí, cambiará el valor predeterminado para que los nuevos dominios utilicen la dirección IPv6 primaria del sistema. Sin embargo, en el momento de la creación, el usuario aún puede seleccionar usar una dirección IPv6 privada o ninguna.
