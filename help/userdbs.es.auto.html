Permitir acceso a bases de datos
Este campo le permite seleccionar a qué bases de datos de este servidor virtual también puede iniciar sesión el usuario. Para agregar una base de datos, selecciónela de la lista de la izquierda y haga clic en la flecha ->.

El nombre de usuario y la contraseña del usuario serán los mismos que usaría para FTP o IMAP. Por defecto, se le otorgará acceso total a todas las tablas en las bases de datos seleccionadas.

MySQL solo admite nombres de usuario de hasta 16 caracteres de longitud, por lo que para un usuario con un nombre de cuenta largo solo se necesitan los primeros 16 caracteres al iniciar sesión en la base de datos.
