Apache kullanıcısı yeni sunucular için Unix grubuna eklensin mi?
Bu seçenek, Apache'nin çalıştığı kullanıcının yeni sanal ana makine grubuna eklenip eklenmeyeceğini belirler. Etkinleştirildiğinde, kullanıcı ana dizinlerinde izinler önemli ölçüde sıkılabilir. Daha sıkı izinler (örn. 750) SuExec ile birleştirildiğinde, etki alanı sahiplerinin diğer etki alanı sahiplerinin dizinlerini ve dosyalarını görmesini imkansız hale getirir. Bu seçenek, ikincil grupları destekleyen tüm platformlarda önerilir.
