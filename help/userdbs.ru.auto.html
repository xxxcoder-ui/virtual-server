Разрешить доступ к базам данных
Это поле позволяет вам выбрать, к каким базам данных, принадлежащим этому виртуальному серверу, пользователь также может подключиться. Чтобы добавить базу данных, выберите ее в левом списке и нажмите стрелку ->.

Логин и пароль пользователя будут такими же, как те, которые он будет использовать для FTP или IMAP. По умолчанию ему будет предоставлен полный доступ ко всем таблицам в выбранных базах данных.

MySQL поддерживает только имена пользователей длиной до 16 символов, поэтому для пользователя с длинным именем учетной записи необходимы только первые 16 символов при входе в базу данных.
