Clone Virtual Server
Questa pagina consente di duplicare un server virtuale esistente con un nuovo nome di dominio. Qualsiasi contenuto Web, record DNS, cassette postali, alias di posta, database e altre impostazioni associate al dominio originale saranno duplicati, ove possibile. Tuttavia, il contenuto Web non verrà modificato in alcun modo, quindi se il server virtuale contiene un'applicazione Web configurata per utilizzare i database o la home directory del vecchio server, dovrà essere riconfigurata manualmente dopo la clonazione.

Quando si duplica un server virtuale di livello superiore, è necessario immettere anche un nuovo nome utente e password per possedere il nuovo dominio. Se il server ha un indirizzo IP privato, verrà automaticamente assegnato un nuovo IP per il clone.

Il processo di clonazione copierà tutti i database associati al dominio originale con nuovi nomi e duplicherà il loro contenuto. Tuttavia, NON aggiornerà gli script installati nel dominio originale che sono configurati per utilizzare i database originali - questo deve essere fatto manualmente.
