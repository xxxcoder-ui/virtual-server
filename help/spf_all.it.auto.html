Azione per altri mittenti
Questo menu determina il modo in cui i server di posta dei destinatari trattano la posta che sembra provenire da questo dominio, ma non da un server consentito specificato utilizzando i campi sopra. Se si è assolutamente certi che tutta la posta elettronica per il dominio passa attraverso questo host di server virtuale o un altro nell'elenco consentito, è necessario selezionare Non consentire .
