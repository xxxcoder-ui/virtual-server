Максимально допустимый псевдоним серверов
В этом поле указывается количество виртуальных серверов с псевдонимами, которые может создать этот владелец сервера. Этот лимит часто используется вместе с Максимально допустимыми серверами без псевдонимов, чтобы дать пользователю разрешения на создание большего или неограниченного количества псевдонимов для его доменов, при этом ограничивая количество реальных виртуальных серверов.
