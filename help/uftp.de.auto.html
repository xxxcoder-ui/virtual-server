FTP-Login aktiviert?
Wenn diese Option auf Ja gesetzt ist, erhält der Benutzer die Shell Damit kann er sich über FTP anmelden. Bei Einstellung auf Nein wird seine Shell auf gesetzt Stattdessen werden FTP-Anmeldungen abgelehnt.

Dies setzt voraus, dass die FTP-Shell in der Datei /etc/shells ist oder auf andere Weise vom auf Ihrem System ausgeführten FTP-Server zugelassen wird.
