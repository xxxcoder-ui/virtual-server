PHP script executie mode
Deze optie bepaald of PHP scripts worden uitgevoerd als de Apache web server gebruiker (typisch httpd of apache), of als de Unix gebruiker waar de virtuele server van is. De eerste optie is sneller omdat er geen extra processen hoeven worden opgestart om de PHP code uit te voeren, maar dit is minder veilig omdat scripts files lezen en schrijven als een gebruiker, anders dan een domein eigenaar.

Indien uw systeem het ondersteund is de FCGId optie het beste, omdat dit een PHP script snel uitvoert als een domein eigenaar. In alle andere gevallen als uw systeem zwaar belast word of veiligheid tussen gebruikers geen probleem is word de CGI wrapper optie aangeraden.
