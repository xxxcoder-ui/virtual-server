SSL-Zertifikat validieren?
Diese Option bestimmt, ob die SSL-Zertifikatsüberprüfung für neue virtuelle Server aktiviert wird, die von Webmin erstellt wurden und für die SSL aktiviert ist. Sie können festlegen, dass nur das Ablaufdatum des Zertifikats überprüft wird, oder überprüfen, ob der Domänenname im Zertifikat mit der URL des virtuellen Servers übereinstimmt.

Wenn das Zertifikat einer Domain innerhalb von 7 Tagen nach Ablauf eingeht, sendet Virtualmin eine Warn-E-Mail an den Eigentümer des virtuellen Servers, damit er Zeit hat, eine Verlängerung zu bestellen.
