Permissies op website directory
Deze optie specificeert de eerste permissies op de files en directory's binnen de gespecificeerde website directory van de vorige optie. Als SuExec is aangezet en de Apache gebruiker is toegevoegd aan de virtuele server gebruiker groep, kunnen deze permissies worden gewijzigd om te voorkomen dat andere gebruikers de inhoud van een website directory kunnen bekijken. Deze strengere permissie set word aanbevolen voor iedereen, behalve wanneer alle gebruikers die shell toegang hebben betrouwbaar zijn. De aanbevolen permissies zijn 750, wat hetzelfde is als rwxr-x---. Met andere woorden, de eigenaar kan lezen, schrijven en uitvoeren, de groep kan lezen en uitvoeren, en andere hebben geen permissie om te lezen, schrijven of files uit te voeren binnen de directory.
