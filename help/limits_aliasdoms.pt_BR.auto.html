Servidores de alias máximos permitidos
Este campo define um limite para o número de servidores virtuais apenas de alias que esse proprietário do servidor pode criar. Esse limite geralmente é usado em conjunto com os servidores não alias máximos permitidos para conceder permissões ao usuário para criar um número maior ou ilimitado de aliases para seus domínios, enquanto ainda limita o número de servidores virtuais reais.
