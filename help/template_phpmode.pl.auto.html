Domyślny tryb wyjątków PHP
Ta opcja określa, czy skrypty PHP na nowych serwerach wirtualnych są uruchamiane jako użytkownik serwera WWW Apache (zazwyczaj httpd lub apache ), czy użytkownik Unix, który jest właścicielem serwera wirtualnego. Ten pierwszy jest szybszy, ponieważ oznacza, że nie trzeba uruchamiać żadnych dodatkowych procesów w celu wykonania kodu PHP, ale mniej bezpieczny, ponieważ skrypty odczytują i zapisują pliki jako użytkownik inny niż właściciel domeny.

Jeśli twój system to obsługuje, opcja FCGId jest najlepsza, ponieważ szybko uruchamia skrypty PHP i jest właścicielem domeny. W przeciwnym razie, chyba że twój system jest mocno obciążony lub bezpieczeństwo między użytkownikami nie stanowi problemu, zaleca się opcję opakowania CGI .
