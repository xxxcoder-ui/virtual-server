Standardbearbeitungsfunktionen
Diese Kontrollkästchen bestimmen, welche Funktionen der Domänenbesitzer ihrer virtuellen Server standardmäßig bearbeiten können. Sobald ein Server erstellt wurde, können diese auf der Seite Eigentümergrenzen bearbeiten geändert werden.

Wenn die Option Automatisch ausgewählt ist, werden die Grenzwerte basierend darauf festgelegt, ob der Eigentümer des virtuellen Servers Subserver erstellen darf oder nicht (gesteuert durch das Feld Anzahl der virtuellen Server begrenzen). In diesem Fall hat er Zugriff auf alle Funktionen. Andernfalls kann er nur Benutzer, Aliase und Webseiten verwalten.
