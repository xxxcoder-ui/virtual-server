Modo de Expiração PHP Padrão
Esta opção determina se os scripts PHP nos novos servidores virtuais são executados como o usuário do servidor web Apache (normalmente httpd ou apache ) ou como o usuário Unix que possui o servidor virtual. O primeiro é mais rápido, pois significa que nenhum processo adicional precisa ser iniciado para executar o código PHP, mas menos seguro, pois os scripts leem e gravam arquivos como um usuário que não seja o proprietário do domínio.

Se o seu sistema suportar, a opção FCGId é a melhor, pois executa scripts PHP rapidamente e como proprietário do domínio. Caso contrário, a menos que seu sistema esteja muito carregado ou que a segurança entre os usuários não seja um problema, a opção Wrapper CGI é recomendada.
