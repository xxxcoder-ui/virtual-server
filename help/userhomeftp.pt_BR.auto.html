Diretório inicial
Por padrão, o diretório inicial de um usuário de FTP do site é o mesmo que o diretório da Web base do domínio. No entanto, você pode alterá-lo para um subdiretório, limitando-o a acessar arquivos somente nesse diretório. Isso pressupõe que o servidor FTP tenha sido configurado para restringir os clientes apenas aos seus diretórios pessoais.
