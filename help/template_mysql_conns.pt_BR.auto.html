Número máximo de conexões para proprietários de domínio
Este campo controla o número de conexões simultâneas ao MySQL que a conta de um proprietário de servidor virtual pode fazer. Isso inclui conexões de scripts instalados pelo Virtualmin, como Wordpress ou Drupal. O padrão é permitir um número ilimitado de conexões, mas como cada uma consome memória no processo do servidor MySQL, você pode limitar o total de conexões em um sistema ocupado.

O limite definido aqui será aplicado somente quando um novo servidor virtual for criado ou quando um login do MySQL estiver ativado para um servidor existente.
