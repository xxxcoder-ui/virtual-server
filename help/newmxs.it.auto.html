Server di posta secondari
Un server di posta secondario può anche ricevere e-mail per uno dei server virtuali gestiti da Virtualmin, ma solo come inoltro temporaneo anziché come destinazione finale. È consigliabile impostare un server secondario per i domini di posta pesantemente utilizzati, per fungere da backup o da conservare nel caso in cui il server di posta primario sia inattivo.

In questa pagina, è possibile selezionare altri server con Virtualmin installato da utilizzare come server MX secondari per i nuovi domini di posta creati su questo sistema. Prima che vengano visualizzati i server, è necessario aggiungerli al modulo Indice server Webmin, in cui è necessario immettere il login e la password di root per il server remoto.

Per ogni server di posta secondario, hai la possibilità di inserire un nome host diverso da utilizzare nel record MX DNS, anziché il nome host effettivo del server, come secmx.tuodominio.com . Tuttavia, assicurati che qualunque cosa tu inserisca sia un nome DNS valido che si risolva sullo stesso server.

Se si fa clic sul pulsante Salva e aggiungi , tutti i server virtuali esistenti per i quali è abilitata la posta elettronica verranno aggiunti a tutti i server di posta secondari selezionati. Inoltre, i record MX DNS verranno aggiunti ai rispettivi domini per fare riferimento ai server di posta.
