Limitar ao modo de demonstração somente leitura?
Se Sim estiver selecionado para esta restrição, o proprietário do servidor não poderá fazer alterações no sistema ou em seus servidores virtuais. No entanto, a interface do usuário do Virtualmin ainda parecerá funcionar exatamente como normal. Isso pode ser útil para criar uma conta demo para mostrar aos clientes em potencial.
