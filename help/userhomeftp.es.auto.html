Directorio de inicio
De manera predeterminada, el directorio de inicio para un usuario FTP del sitio web es el mismo que el directorio web base del dominio. Sin embargo, puede cambiarlo para que sea un subdirectorio, para limitarlo a acceder a los archivos solo en ese directorio.
