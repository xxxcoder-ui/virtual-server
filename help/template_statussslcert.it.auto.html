Convalida certificato SSL?
Questa opzione determina se la convalida del certificato SSL sarà abilitata per i nuovi server virtuali creati da Webmin con SSL abilitato. È possibile scegliere di controllare solo la data di scadenza del certificato o di convalidare che il nome di dominio nel certificato corrisponda all'URL del server virtuale.

Quando il certificato di un dominio arriva entro 7 giorni dalla scadenza, Virtualmin invierà un'e-mail di avviso al proprietario del server virtuale dandogli il tempo di ordinare un rinnovo.
