Sottoprocessi PHP FCGId predefiniti
Quando gli script PHP vengono eseguiti tramite FCGId, ogni dominio avrà il numero di processi PHP impostati in questo campo sempre in esecuzione per soddisfare le richieste. Puoi aumentarlo dal valore predefinito di 4 per migliorare la latenza degli script PHP o ridurlo per ridurre l'uso della memoria.

Impostandolo su Nessuno , i processi PHP verranno avviati solo se necessario su richiesta e verranno ripuliti dopo un certo periodo di inattività.
