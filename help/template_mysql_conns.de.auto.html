Maximale Verbindungen für Domaininhaber
Dieses Feld steuert die Anzahl der gleichzeitigen Verbindungen zu MySQL, die das Konto eines Besitzers eines virtuellen Servers herstellen kann. Dies schließt Verbindungen von Skripten ein, die von Virtualmin installiert wurden, z. B. Wordpress oder Drupal. Standardmäßig ist eine unbegrenzte Anzahl von Verbindungen zulässig. Da jedoch jede innerhalb des MySQL-Serverprozesses Speicher belegt, möchten Sie möglicherweise die Gesamtzahl der Verbindungen auf einem ausgelasteten System begrenzen.

Das hier festgelegte Limit wird nur angewendet, wenn ein neuer virtueller Server erstellt wird oder wenn eine MySQL-Anmeldung für einen vorhandenen Server aktiviert ist.
