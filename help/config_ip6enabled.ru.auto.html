Использовать IPv6-адрес по умолчанию для новых виртуальных серверов?
По умолчанию (если для этого параметра установлено значение «Нет» ) Virtualmin не будет добавлять записи DNS или конфигурацию веб-сервера на новые виртуальные серверы для системного IPv6-адреса по умолчанию. Однако выбор «Да» здесь изменит значение по умолчанию для новых доменов, использующих основной IPv6-адрес системы. Во время создания пользователь все еще может выбрать использование личного адреса IPv6 или его отсутствие вообще.
