Pode escolher nomes de banco de dados?
Se essa restrição estiver definida como Não , o proprietário do servidor não poderá selecionar o nome do banco de dados padrão ao criar um sub-servidor com um recurso de banco de dados ativado. Em vez disso, sempre será o padrão para a primeira parte do nome de domínio.

Além disso, ele não poderá remover o banco de dados padrão de nenhum domínio existente, pois isso permitiria renomear efetivamente o banco de dados padrão após a criação do domínio.
