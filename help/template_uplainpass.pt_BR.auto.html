Armazenar senhas de texto não criptografado?
Quando essa opção está definida como Sim , o Virtualmin armazenará as senhas de texto não criptografado para proprietários de domínio e caixas de correio. Isso permite que eles sejam exibidos na interface do usuário, mas adiciona um risco potencial à segurança se o seu sistema estiver comprometido. Selecionar Não fará com que todos os novos servidores virtuais criados a partir deste modelo armazenem apenas senhas de hash unidirecionais.

Como os aplicativos da Web precisam armazenar senhas do banco de dados em texto sem formatação, a senha do banco de dados de um usuário será definida como uma nova senha aleatória se o Virtualmin estiver configurado para não armazenar senhas de texto não criptografado. Você pode visualizar essa senha acessando Editar bancos de dados - <Senhas.
