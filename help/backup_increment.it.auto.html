Livello di backup
Per impostazione predefinita, i backup di Virtualmin includeranno tutti i file nelle directory home dei domini selezionati. Tuttavia, se si esegue regolarmente il backup di server virtuali di grandi dimensioni, potrebbe essere impiegato molto tempo a copiare file che non sono stati modificati di recente.

Se l'opzione Incrementale è selezionata qui, verranno inclusi solo i file che sono stati modificati o creati dall'ultimo backup completo. Ciò è particolarmente utile quando si hanno due backup pianificati, come un backup completo settimanale e un incremento giornaliero. Supponendo che la maggior parte dei file in un dominio non vengano modificati giorno per giorno, ciò ridurrà sostanzialmente la dimensione e l'ora dei backup giornalieri.

Tenere presente che durante il ripristino, è necessario ripristinare prima il backup completo e poi quello incrementale per ripristinare tutti i file. Inoltre, non viene eseguito il backup incrementale di altri dati come le configurazioni e i database di Apache, pertanto verranno sempre inclusi indipendentemente da questa opzione.

L'opzione Nessuno dei due esegue anche un backup completo, ma non modifica lo stato del backup incrementale. Ciò può essere utile quando si esegue un backup ad hoc di domini che hanno pianificato anche backup completi e incrementali.
