Subprocessos PHP FCGId padrão
Quando os scripts PHP são executados via FCGId, cada domínio terá o número de processos PHP definidos neste campo em execução o tempo todo para atender solicitações. Você pode aumentar isso do padrão 4 para melhorar a latência do script PHP ou diminuí-la para reduzir o uso de memória.

Configurá-lo como None fará com que os processos PHP sejam iniciados apenas conforme necessário sob demanda e sejam limpos após algum período de inatividade.
