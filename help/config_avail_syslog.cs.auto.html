Systémové protokoly (zobrazit protokoly Apache a FTP)
Pokud je tato možnost povolena, budou moci administrátoři virtuálního serveru prohlížet obsah svých souborů Apache a virtuálních anonymních souborů protokolu protokolu FTP prostřednictvím webového rozhraní. To se provádí udělením omezeného přístupu k modulu System Logs.
