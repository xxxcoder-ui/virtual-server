Acción para otros remitentes
Este menú determina cómo los servidores de correo del destinatario tratan el correo electrónico que parece provenir de este dominio, pero que no proviene de un servidor permitido especificado utilizando los campos anteriores. Si está absolutamente seguro de que todo el correo electrónico para el dominio pasa a través de este servidor virtual u otro en la lista permitida, debe seleccionar No permitir.
