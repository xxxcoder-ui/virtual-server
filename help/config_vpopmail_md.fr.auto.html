Sous-répertoire VPOPMail pour le courrier électronique
Ce champ vous permet de personnaliser le sous-répertoire sous le domicile de chaque utilisateur VPOPMail qui contiendra le courrier électronique. Sur les systèmes qui n'utilisent pas la valeur par défaut de Maildir , cela devra être modifié pour correspondre au chemin réel utilisé par VPOPMail - par exemple, sous Gentoo Linux .maildir est utilisé.
